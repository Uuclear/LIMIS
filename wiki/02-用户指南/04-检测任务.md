# 检测任务

本文档介绍检测任务的管理和执行流程。

---

## 一、功能概述

检测任务是分配给检测人员的具体检测工作，主要功能包括：

- **任务查看**：查看分配给自己的检测任务
- **任务接收**：确认接收检测任务
- **任务执行**：执行检测并填写结果
- **任务提交**：提交检测结果

### 1.1 适用角色

| 角色 | 权限 |
|------|------|
| 检测人员 | 接收、执行、提交任务 |
| 技术负责人 | 分配任务、监控进度 |
| 样品管理员 | 查看任务状态 |

### 1.2 任务状态流转

```
┌──────────┐   分配    ┌──────────┐   接收    ┌──────────┐
│  待分配  │ ───────▶ │  待检    │ ───────▶ │ 检测中   │
│(assigned)│          │(pending) │          │(progress)│
└──────────┘          └──────────┘          └──────────┘
                            │                     │
                            │ 退回                │ 完成
                            ▼                     ▼
                       ┌──────────┐         ┌──────────┐
                       │  异常    │         │  已完成  │
                       │(abnormal)│         │(completed)│
                       └──────────┘         └──────────┘
```

## 二、任务列表

### 2.1 查看任务列表

**路径**：检测业务 → 检测任务

任务列表显示分配给当前用户的检测任务。

### 2.2 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 任务编号 | 模糊搜索 |
| 样品编号 | 盲样编号搜索 |
| 检测类型 | 下拉选择 |
| 状态 | 待检/检测中/已完成/异常 |
| 截止日期 | 日期范围 |

### 2.3 列表字段

| 字段 | 说明 |
|------|------|
| 任务编号 | 任务唯一标识 |
| 盲样编号 | 样品盲样编号 |
| 检测项目 | 检测参数名称 |
| 检测方法 | 执行标准 |
| 状态 | 当前状态 |
| 截止日期 | 要求完成日期 |
| 剩余天数 | 距截止日期天数 |

## 三、任务接收

### 3.1 查看任务详情

点击任务编号进入详情页，查看：
- 样品信息（盲样编号、名称、规格）
- 检测要求（检测项目、方法、标准）
- 设备要求（需使用的设备）
- 截止日期

### 3.2 接收任务

1. 确认可以完成任务
2. 点击「接收任务」按钮
3. 任务状态变更为「检测中」

### 3.3 拒绝任务

如无法完成任务：
1. 点击「退回」按钮
2. 填写退回原因
3. 任务返回待分配状态

---

## 四、任务执行

### 4.1 准备工作

执行检测前需确认：
- [ ] 样品状态正常
- [ ] 设备已校准且在有效期内
- [ ] 环境条件满足要求
- [ ] 已阅读检测方法标准

### 4.2 设备使用登记

1. 在任务详情页点击「设备使用登记」
2. 选择使用的设备
3. 记录使用时间
4. 系统自动关联设备使用日志

### 4.3 环境条件记录

检测前需记录环境条件：
- 温度（°C）
- 相对湿度（%）

系统会自动校验是否在允许范围内。

## 五、原始记录填写

### 5.1 进入记录填写页面

**路径**：检测业务 → 原始记录 → 新建

或从任务详情页点击「填写记录」。

### 5.2 选择记录模板

系统根据检测项目自动匹配记录模板，也可手动选择。

### 5.3 填写记录内容

#### 基本信息

| 字段 | 说明 |
|------|------|
| 任务编号 | 自动关联 |
| 样品编号 | 盲样编号 |
| 检测日期 | 默认当天 |
| 环境温度 | 检测环境温度 |
| 环境湿度 | 检测环境湿度 |

#### 检测数据

根据模板填写检测数据，常见字段：

**混凝土抗压强度**：
| 字段 | 单位 | 说明 |
|------|------|------|
| 试件尺寸 | mm | 150×150×150 |
| 受压面积 | mm² | 自动计算 |
| 破坏荷载 | kN | 测量值 |
| 抗压强度 | MPa | 自动计算 |

**钢筋拉伸试验**：
| 字段 | 单位 | 说明 |
|------|------|------|
| 公称直径 | mm | 钢筋直径 |
| 横截面积 | mm² | 自动计算 |
| 屈服荷载 | kN | 测量值 |
| 最大荷载 | kN | 测量值 |
| 屈服强度 | MPa | 自动计算 |
| 抗拉强度 | MPa | 自动计算 |
| 断后伸长率 | % | 测量计算 |

### 5.4 自动计算

系统内置公式库，支持自动计算：

| 检测类型 | 计算公式 |
|----------|----------|
| 混凝土抗压强度 | fc = F × 1000 / A |
| 钢筋屈服强度 | ReL = Fy × 1000 / S |
| 钢筋抗拉强度 | Rm = Fm × 1000 / S |
| 含水率 | w = (m - md) / md × 100 |
| 压实度 | K = ρd / ρd,max × 100 |

### 5.5 结果判定

系统根据判定规则自动判定结果：

| 判定类型 | 说明 |
|----------|------|
| 数值判定 | 与标准值比较 |
| 范围判定 | 是否在允许范围内 |
| 等级判定 | 根据强度等级判定 |

### 5.6 保存记录

- **保存草稿**：保存但不提交，可继续编辑
- **提交复核**：提交给复核人审核

## 六、记录修改

### 6.1 修改条件

只有**草稿状态**的记录可以自由修改。

已提交的记录如需修改：
1. 联系复核人退回
2. 退回后可修改
3. 修改会留下痕迹

### 6.2 修改痕迹

所有修改都会记录：
- 修改字段
- 原值
- 新值
- 修改人
- 修改时间
- 修改原因

可在记录详情页的「修改痕迹」标签页查看。

---

## 七、任务完成

### 7.1 提交检测结果

1. 确认原始记录已填写完整
2. 确认检测结果已判定
3. 点击「提交结果」按钮
4. 任务状态变更为「已完成」

### 7.2 关联报告

完成的任务会自动关联到报告编制环节。

---

## 八、任务超期处理

### 8.1 超期预警

- 截止日期前3天：黄色标记
- 截止日期当天：橙色标记
- 超过截止日期：红色标记

### 8.2 超期处理

如任务超期：
1. 尽快完成检测
2. 在备注中说明超期原因
3. 系统记录超期信息

## 九、常见问题

### Q1: 如何查看样品的真实编号？

**答案**：检测人员只能看到盲样编号，无法查看真实编号。这是为了保证检测公正性。

### Q2: 设备校准过期了还能用吗？

**答案**：不可以。系统会阻止使用校准过期的设备，需联系设备管理员安排校准。

### Q3: 原始记录填错了怎么办？

**答案**：
- 草稿状态：直接修改
- 已提交状态：联系复核人退回后修改

### Q4: 如何查看检测方法标准？

**答案**：在任务详情页可以查看检测方法关联的标准文件。

### Q5: 任务可以转给其他人吗？

**答案**：可以。联系技术负责人重新分配任务。
