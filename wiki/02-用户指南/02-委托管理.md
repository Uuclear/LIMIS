# 委托管理

本文档介绍委托管理的完整操作流程，包括委托登记、提交、评审等环节。

---

## 一、功能概述

委托管理是检测业务的起点，主要功能包括：

- **委托登记**：创建新的检测委托单
- **委托编辑**：修改草稿状态的委托单
- **委托提交**：提交委托单进入评审流程
- **合同评审**：对委托进行技术能力和资源评估
- **委托查询**：查询和查看委托单信息

### 1.1 适用角色

| 角色 | 权限 |
|------|------|
| 接待员 | 创建、编辑、提交委托 |
| 技术负责人 | 合同评审 |
| 系统管理员 | 全部操作 |

### 1.2 委托状态流转

```
┌─────────┐    提交    ┌───────────┐   评审通过   ┌─────────┐
│  草稿   │ ────────▶ │ 待评审    │ ──────────▶ │ 已评审  │
│ (draft) │           │(pending)  │             │(reviewed)│
└─────────┘           └───────────┘             └─────────┘
     │                     │                         
     │ 取消                │ 评审退回                
     ▼                     ▼                         
┌─────────┐           ┌─────────┐                   
│ 已取消  │           │ 已退回  │ ──▶ 修改后重新提交
│(cancelled)│         │(rejected)│                  
└─────────┘           └─────────┘                   
```

## 二、委托登记

### 2.1 进入委托登记页面

**路径**：委托管理 → 新增委托

或从委托列表页点击「新增」按钮。

### 2.2 填写委托基本信息

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 委托编号 | 自动 | 系统自动生成，格式：WT-YYYYMMDD-XXX |
| 工程项目 | 是 | 选择关联的工程项目 |
| 委托单位 | 是 | 选择委托单位（参建单位） |
| 委托日期 | 是 | 默认当天，可修改 |
| 联系人 | 是 | 委托方联系人姓名 |
| 联系电话 | 是 | 联系人电话 |
| 委托类型 | 是 | 常规/加急/特殊 |
| 要求完成日期 | 是 | 委托方要求的报告完成日期 |
| 见证人 | 否 | 选择见证人（如有） |
| 备注 | 否 | 其他说明信息 |

### 2.3 添加委托项目

委托项目是委托单的检测内容明细，一个委托单可包含多个检测项目。

点击「添加检测项目」按钮，填写以下信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 检测对象 | 是 | 选择检测对象类型（混凝土/钢筋/沥青/土工等） |
| 检测项目 | 是 | 选择具体检测参数 |
| 检测标准 | 是 | 选择执行标准 |
| 规格型号 | 是 | 材料规格，如C30、HRB400 |
| 样品数量 | 是 | 样品数量 |
| 样品编号 | 否 | 如已有样品编号可填写 |
| 检测依据 | 是 | 检测方法依据 |

**示例**：

| 检测对象 | 检测项目 | 检测标准 | 规格型号 | 数量 |
|----------|----------|----------|----------|------|
| 混凝土 | 抗压强度 | GB/T 50081 | C30 | 3组 |
| 钢筋 | 屈服强度 | GB/T 228.1 | HRB400 Φ16 | 2根 |
| 钢筋 | 抗拉强度 | GB/T 228.1 | HRB400 Φ16 | 2根 |

### 2.4 上传附件

可上传以下附件：
- 委托单扫描件
- 取样记录
- 见证记录
- 其他相关文件

**支持格式**：PDF、JPG、PNG、DOC、DOCX
**单个文件大小限制**：10MB

### 2.5 保存委托

- **保存草稿**：保存但不提交，可继续编辑
- **保存并提交**：保存后直接进入评审流程

## 三、委托编辑

### 3.1 编辑条件

只有**草稿状态**的委托单可以编辑。

### 3.2 编辑步骤

1. 进入委托列表页
2. 找到目标委托单（状态为"草稿"）
3. 点击「编辑」按钮
4. 修改需要更新的信息
5. 点击「保存」

### 3.3 注意事项

- 已提交的委托单不能直接编辑
- 如需修改已提交的委托，需先退回或取消

---

## 四、委托提交

### 4.1 提交条件

- 委托信息填写完整
- 至少添加一个检测项目
- 无必填字段为空

### 4.2 提交步骤

1. 在委托编辑页面点击「提交」
2. 系统校验信息完整性
3. 校验通过后，状态变更为「待评审」
4. 系统自动通知技术负责人进行评审

### 4.3 提交后操作

提交后委托单进入评审流程，此时：
- 不能再编辑委托信息
- 可查看委托详情
- 可跟踪评审进度

---

## 五、合同评审

合同评审是对委托进行技术能力和资源评估的过程。

### 5.1 评审入口

**路径**：委托管理 → 合同评审

或从首页「待办事项」进入。

### 5.2 评审内容

评审人员需评估以下内容：

| 评审项目 | 说明 |
|----------|------|
| 资质能力 | 实验室是否具备相应检测资质 |
| 设备能力 | 是否有相应的检测设备且在有效期内 |
| 人员能力 | 是否有授权的检测人员 |
| 标准有效性 | 检测标准是否现行有效 |
| 工期评估 | 能否在要求日期内完成 |
| 资源评估 | 现有资源是否满足检测需求 |

### 5.3 评审结论

| 结论 | 说明 | 后续操作 |
|------|------|----------|
| 接受 | 满足全部要求 | 委托进入样品登记环节 |
| 有条件接受 | 部分条件需协商 | 与委托方沟通后接受 |
| 拒绝 | 不满足要求 | 委托退回，说明原因 |

### 5.4 评审操作

1. 查看委托详情和检测项目
2. 填写评审意见
3. 选择评审结论
4. 点击「提交评审」

**评审通过**：
- 委托状态变更为「已评审」
- 系统自动生成样品编号
- 通知样品管理员进行样品登记

**评审退回**：
- 委托状态变更为「已退回」
- 接待员可修改后重新提交

## 六、委托查询

### 6.1 列表筛选

委托列表页支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 委托编号 | 模糊搜索 |
| 工程项目 | 下拉选择 |
| 委托单位 | 下拉选择 |
| 状态 | 草稿/待评审/已评审/已退回/已取消 |
| 委托日期 | 日期范围 |
| 要求完成日期 | 日期范围 |

### 6.2 列表字段

| 字段 | 说明 |
|------|------|
| 委托编号 | 唯一标识 |
| 工程项目 | 关联项目名称 |
| 委托单位 | 委托方名称 |
| 委托日期 | 登记日期 |
| 检测项目数 | 委托项目数量 |
| 状态 | 当前状态 |
| 操作 | 查看/编辑/取消等 |

### 6.3 详情查看

点击委托编号或「查看」按钮，进入委托详情页。

详情页包含以下标签页：
- **基本信息**：委托基本信息
- **检测项目**：委托项目明细
- **样品信息**：关联的样品列表
- **附件**：上传的附件文件
- **操作日志**：状态变更记录

## 七、委托取消

### 7.1 取消条件

只有**草稿状态**的委托可以取消。

### 7.2 取消操作

1. 在委托列表找到目标委托
2. 点击「取消」按钮
3. 填写取消原因
4. 确认取消

### 7.3 取消后

- 委托状态变更为「已取消」
- 不能再恢复或编辑
- 如需重新委托，需新建委托单

## 八、委托打印

### 8.1 打印委托单

1. 进入委托详情页
2. 点击「打印」按钮
3. 选择打印模板
4. 预览并打印

### 8.2 打印内容

打印的委托单包含：
- 委托编号
- 委托单位信息
- 检测项目明细
- 取样信息
- 签字栏

## 九、常见问题

### Q1: 委托编号可以修改吗？

**答案**：不可以。委托编号由系统自动生成，确保唯一性。

### Q2: 已提交的委托如何修改？

**解决方法**：
1. 联系技术负责人退回委托
2. 退回后可重新编辑
3. 修改完成后重新提交

### Q3: 评审退回的委托还能再提交吗？

**答案**：可以。修改完成后可重新提交进入评审流程。

### Q4: 如何查看委托的评审意见？

**方法**：进入委托详情页，查看「操作日志」标签页，可看到评审记录和意见。

### Q5: 委托的项目可以部分接受吗？

**答案**：可以。评审时可以修改检测项目，删除不接受的项目，或标注有条件接受的项目。
