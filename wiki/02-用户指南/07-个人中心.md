# 个人中心

本文档介绍个人中心的功能，包括个人信息管理、密码修改、电子签名和通知管理。

---

## 一、功能概述

个人中心提供用户个人信息管理功能：

- **个人信息**：查看和修改个人信息
- **密码修改**：修改登录密码
- **电子签名**：上传和管理电子签名
- **通知管理**：查看系统通知

---

## 二、个人信息

### 2.1 查看个人信息

**入口**：点击右上角用户头像 → 个人中心

或点击侧边栏「个人中心」。

### 2.2 个人信息内容

| 字段 | 说明 | 可修改 |
|------|------|:------:|
| 用户名 | 登录用户名 | 否 |
| 姓名 | 真实姓名 | 是 |
| 手机号 | 联系电话 | 是 |
| 邮箱 | 电子邮箱 | 是 |
| 部门 | 所属部门 | 是 |
| 职称 | 职称职务 | 是 |
| 头像 | 个人头像 | 是 |

### 2.3 修改个人信息

1. 点击「编辑」按钮
2. 修改需要更新的字段
3. 点击「保存」

### 2.4 上传头像

1. 点击头像区域
2. 选择图片文件
3. 支持格式：JPG、PNG
4. 文件大小：不超过2MB
5. 系统自动裁剪为正方形

## 三、密码修改

### 3.1 修改密码入口

**路径**：个人中心 → 密码修改

或点击右上角用户头像 → 修改密码。

### 3.2 修改密码步骤

1. 输入**当前密码**
2. 输入**新密码**
3. 确认**新密码**
4. 点击「确认修改」

### 3.3 密码要求

| 要求 | 说明 |
|------|------|
| 长度 | 至少8个字符 |
| 复杂度 | 包含大小写字母、数字、特殊字符中的至少3种 |
| 历史密码 | 不能与最近5次密码相同 |
| 有效期 | 建议90天更换一次 |

### 3.4 密码安全建议

- ✅ 使用强密码（字母+数字+符号组合）
- ✅ 定期更换密码
- ✅ 不同系统使用不同密码
- ❌ 不要使用生日、电话号码等易猜密码
- ❌ 不要与他人共享密码
- ❌ 不要将密码写在纸上或保存在电脑中

### 3.5 忘记密码

如忘记密码，请联系系统管理员重置。

管理员重置后，您将收到临时密码，登录后需立即修改。

## 四、电子签名

### 4.1 电子签名概述

电子签名用于报告签署，具有法律效力。授权签字人批准报告时，系统会自动将签名添加到报告中。

### 4.2 上传签名

**路径**：个人中心 → 电子签名

#### 签名要求

| 要求 | 说明 |
|------|------|
| 格式 | PNG、JPG |
| 大小 | 不超过2MB |
| 背景 | 透明或白色 |
| 清晰度 | 签名清晰可辨 |
| 尺寸 | 建议200×100像素 |

#### 上传步骤

1. 点击「上传签名」按钮
2. 选择签名图片文件
3. 系统验证格式和大小
4. 预览签名效果
5. 确认上传

### 4.3 签名预览

上传成功后可预览签名效果：
- 显示签名图片
- 显示上传时间
- 显示签名状态

### 4.4 更换签名

如需更换签名：
1. 点击「删除签名」按钮
2. 重新上传新签名
3. 新签名立即生效

### 4.5 签名使用

批准报告时：
- 系统自动检查是否有有效签名
- 无签名时提示先上传
- 有签名时自动添加到报告

### 4.6 签名权限

只有**授权签字人**角色可以使用电子签名签署报告。

其他角色上传的签名仅用于个人标识，不用于报告签署。

## 五、通知管理

### 5.1 查看通知

**入口**：点击顶部导航栏的通知图标

或**路径**：个人中心 → 我的通知

### 5.2 通知类型

| 类型 | 说明 |
|------|------|
| 委托待评审 | 有新的委托单需要评审 |
| 设备即将到期 | 设备校准即将到期 |
| 样品超期 | 样品龄期已超期 |
| 任务已分配 | 有新的检测任务 |
| 记录待复核 | 有原始记录待复核 |
| 报告待审核 | 有报告待审核 |
| 报告待批准 | 有报告待批准 |
| 质量审核 | 质量相关通知 |
| 系统通知 | 系统公告 |

### 5.3 通知操作

| 操作 | 说明 |
|------|------|
| 查看 | 点击通知跳转到相关页面 |
| 标记已读 | 将单条通知标记为已读 |
| 全部已读 | 将所有通知标记为已读 |
| 删除 | 删除通知（仅删除显示，不删除记录） |

### 5.4 通知设置

可设置接收哪些类型的通知：
1. 进入个人中心
2. 点击「通知设置」
3. 勾选需要接收的通知类型
4. 保存设置

### 6.1 查看待办事项

**路径**：个人中心 → 我的待办

或从首页「待办事项」卡片进入。

4. 完成处理后，待办自动消失

## 八、常见问题

### Q1: 用户名可以修改吗？

**答案**：不可以。用户名是登录凭证，创建后不能修改。

### Q2: 密码忘记了怎么办？

**答案**：联系系统管理员重置密码。

### Q3: 电子签名上传失败怎么办？

**排查步骤**：
1. 检查文件格式是否为PNG或JPG
2. 检查文件大小是否超过2MB
3. 检查图片是否清晰
4. 尝试更换浏览器

### Q4: 为什么收不到通知？

**排查步骤**：
1. 检查通知设置是否开启
2. 检查是否有对应权限
3. 检查浏览器通知权限

### Q5: 如何注销账号？

**答案**：账号不能自行注销，如需停用请联系系统管理员。

## 九、安全建议

### 9.1 账号安全

- 定期修改密码（建议90天）
- 使用强密码
- 不要共享账号
- 发现异常及时报告

### 9.2 签名安全

- 妥善保管签名图片
- 不要将签名借给他人使用
- 发现签名被盗用及时报告

### 9.3 数据安全

- 不要在公共电脑保存密码
- 使用完毕及时退出登录
- 不要将敏感信息截图外传
