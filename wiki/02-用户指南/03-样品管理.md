# 样品管理

本文档介绍样品管理的完整操作流程，包括样品登记、流转、留样和处置等环节。

---

## 一、功能概述

样品管理是检测业务的核心环节，主要功能包括：

- **样品登记**：创建新的样品记录
- **盲样管理**：分配盲样编号，保护委托方信息
- **样品流转**：管理样品状态变更
- **留样管理**：检测完成后的样品保留
- **样品处置**：留样到期后的处理

### 1.1 适用角色

| 角色 | 权限 |
|------|------|
| 样品管理员 | 样品登记、流转、处置 |
| 检测人员 | 查看样品（盲样编号） |
| 技术负责人 | 查看样品、盲样映射 |

### 1.2 样品状态流转

```
┌─────────┐   收样    ┌─────────┐   分配    ┌─────────┐
│  待检   │ ───────▶ │ 检测中  │ ───────▶ │  已检   │
│(pending)│          │(testing)│          │ (tested)│
└─────────┘          └─────────┘          └─────────┘
     │                                          │
     │ 直接退还                                  │ 留样
     ▼                                          ▼
┌─────────┐                                ┌─────────┐
│ 已退还  │                                │  留样   │
│(returned)│                               │(retained)│
└─────────┘                                └─────────┘
                                                 │
                                                 │ 到期处置
                                                 ▼
                                           ┌─────────┐
                                           │ 已处置  │
                                           │(disposed)│
                                           └─────────┘
```

## 二、样品登记

### 2.1 单个样品登记

**路径**：样品管理 → 样品登记

#### 填写样品信息

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 样品编号 | 自动 | 系统自动生成，格式：YP-YYYYMMDD-XXX |
| 盲样编号 | 自动 | 可手动分配或系统自动生成 |
| 样品名称 | 是 | 样品名称 |
| 委托单 | 是 | 关联的委托单 |
| 样品类型 | 是 | 混凝土/钢筋/沥青/土工等 |
| 规格型号 | 是 | 材料规格 |
| 设计强度/等级 | 否 | 如C30、HRB400 |
| 数量 | 是 | 样品数量 |
| 单位 | 是 | 个/组/根/kg等 |
| 取样日期 | 是 | 样品取样日期 |
| 取样地点 | 否 | 取样位置描述 |
| 收样日期 | 是 | 默认当天 |
| 送样人 | 否 | 送样人员姓名 |
| 见证人 | 否 | 见证取样人员 |
| 备注 | 否 | 其他说明 |

#### 样品标签打印

登记完成后可打印样品标签，标签包含：
- 样品编号（或盲样编号）
- 样品名称
- 规格型号
- 收样日期
- 二维码

### 2.2 批量样品登记

适用于同批次多个样品的快速登记。

**操作步骤**：
1. 点击「批量登记」按钮
2. 选择委托单
3. 设置样品数量和基本信息
4. 系统自动生成多个样品记录
5. 可单独修改每个样品的特定信息

### 2.3 从委托单生成样品

评审通过的委托单可自动生成样品：

1. 进入委托详情页
2. 点击「生成样品」按钮
3. 系统根据委托项目自动创建样品
4. 样品信息自动关联委托单

## 三、盲样管理

### 3.1 盲样概念

盲样是隐藏委托方信息的样品编号，用于保证检测公正性。检测人员只能看到盲样编号，无法获知样品来源。

### 3.2 分配盲样编号

**单个分配**：
1. 进入样品详情页
2. 点击「分配盲样编号」按钮
3. 系统自动生成盲样编号

**批量分配**：
1. 在样品列表勾选多个样品
2. 点击「批量分配盲样编号」
3. 系统为所有选中样品生成盲样编号

**盲样编号格式**：BL-YYYYMM-XXXXXX
- BL：盲样前缀
- YYYYMM：年月
- XXXXXX：6位随机字符

### 3.3 盲样映射表

盲样映射表记录盲样编号与真实编号的对应关系。

**查看权限**：
- 系统管理员
- 技术负责人
- 质量负责人
- 样品管理员
- 设备管理员

**查看路径**：样品管理 → 盲样映射表

### 3.4 检测人员使用盲样

检测人员在操作时：
- 只能看到盲样编号
- 无法查看委托方信息
- 无法查看工程项目信息
- 检测结果与盲样编号关联

## 四、样品流转

### 4.1 状态变更

样品状态变更需按规定的流转路径进行：

| 当前状态 | 可变更状态 | 操作说明 |
|----------|------------|----------|
| 待检 | 检测中 | 分配检测任务时自动变更 |
| 待检 | 已退还 | 直接退还样品 |
| 检测中 | 已检 | 检测完成时自动变更 |
| 已检 | 留样 | 检测完成后留样 |
| 留样 | 已处置 | 留样到期处置 |
| 留样 | 已退还 | 留样退还委托方 |

### 4.2 状态变更操作

1. 进入样品详情页
2. 点击「变更状态」按钮
3. 选择目标状态
4. 填写变更原因（必填）
5. 确认变更

### 4.3 状态变更记录

所有状态变更都会记录：
- 变更时间
- 变更人
- 原状态
- 新状态
- 变更原因

可在样品详情页的「流转记录」标签页查看。

## 五、留样管理

### 5.1 留样登记

检测完成后，需要保留的样品进行留样登记：

1. 进入样品详情页
2. 点击「留样登记」按钮
3. 填写留样信息：

| 字段 | 说明 |
|------|------|
| 留样位置 | 存放位置，如样品室A-货架1-层3 |
| 留样期限 | 保留天数，如28天 |
| 保存条件 | 温度、湿度等要求 |
| 备注 | 其他说明 |

4. 确认留样

### 5.2 留样查询

**路径**：样品管理 → 留样管理

可查询：
- 当前留样列表
- 即将到期留样
- 已过期留样

### 5.3 留样到期提醒

系统自动检测留样到期情况：
- 到期前3天：发送提醒通知
- 到期当天：标记为待处置
- 过期：标记为超期

## 六、样品处置

### 6.1 处置方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 退还 | 退还给委托方 | 有价值的样品 |
| 销毁 | 专业销毁处理 | 危险品、化学品 |
| 丢弃 | 普通废弃处理 | 无害普通样品 |

### 6.2 处置操作

1. 进入留样管理页面
2. 选择待处置的样品
3. 点击「处置」按钮
4. 填写处置信息：

| 字段 | 说明 |
|------|------|
| 处置方式 | 退还/销毁/丢弃 |
| 处置日期 | 实际处置日期 |
| 处置人 | 执行处置的人员 |
| 接收人 | 退还时的接收人（如适用） |
| 备注 | 其他说明 |

5. 确认处置

### 6.3 处置记录

所有处置记录保存在系统中，可查询和导出。

### 8.2 标签打印

1. 在样品列表勾选需要打印的样品
2. 点击「打印标签」按钮
3. 选择标签模板
4. 设置打印数量
5. 打印

### 8.3 标签粘贴

- 粘贴在样品容器或包装上
- 确保标签清晰可读
- 避免标签污损或脱落

## 九、龄期管理

### 9.1 龄期概念

龄期是指混凝土等材料从成型到检测的时间间隔。常见龄期：
- 3天
- 7天
- 28天（标准龄期）
- 60天
- 90天

### 9.2 龄期设置

在样品登记时可设置龄期要求：
1. 选择龄期天数
2. 系统自动计算到期日期
3. 到期前发送提醒

### 9.3 龄期提醒

- 到期前3天：发送提醒
- 到期当天：标记为待检测
- 过期：标记为超期

## 十、常见问题

### Q1: 样品编号可以修改吗？

**答案**：不可以。样品编号由系统自动生成，确保唯一性。

### Q2: 如何查看盲样对应的真实编号？

**答案**：需要有相应权限（样品管理员、技术负责人等），在「盲样映射表」中查看。

### Q3: 样品状态变更错误怎么办？

**答案**：联系系统管理员处理。部分状态变更可以回退。

### Q4: 留样到期后忘记处置会怎样？

**答案**：系统会持续发送提醒通知，样品状态显示为"超期"。

### Q5: 如何批量导出样品信息？

**答案**：在样品列表页点击「导出」按钮，选择导出字段和格式。
