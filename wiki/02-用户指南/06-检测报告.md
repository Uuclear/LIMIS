# 检测报告

本文档介绍检测报告的查看、审批和发放流程。

---

## 一、功能概述

检测报告是检测工作的最终成果，主要功能包括：

- **报告查看**：查看检测报告内容和状态
- **报告审批**：审核、批准报告
- **报告签署**：电子签名签署
- **报告发放**：报告发放登记

### 1.1 适用角色

| 角色 | 权限 |
|------|------|
| 检测人员 | 查看已批准的报告 |
| 审核人 | 审核报告 |
| 授权签字人 | 批准、签署报告 |
| 接待员 | 报告发放 |
| 委托方 | 查看自己的报告 |

### 1.2 报告状态流转

```
┌─────────┐   提交    ┌──────────┐   审核通过  ┌──────────┐
│  草稿   │ ───────▶ │ 待审核   │ ─────────▶ │ 待批准   │
│ (draft) │          │(pending) │            │(approval)│
└─────────┘          └──────────┘            └──────────┘
                          │                        │
                          │ 退回                   │ 批准
                          ▼                        ▼
                     ┌─────────┐             ┌─────────┐
                     │ 已退回  │             │ 已批准  │
                     │(returned)│            │(approved)│
                     └─────────┘             └─────────┘
                                                  │
                                                  │ 发放
                                                  ▼
                                             ┌─────────┐
                                             │ 已发放  │
                                             │(issued) │
                                             └─────────┘
                                                  │
                                                  │ 归档
                                                  ▼
                                             ┌─────────┐
                                             │ 已归档  │
                                             │(archived)│
                                             └─────────┘

                    任意状态 ──────────────────▶ 已作废(voided)
```

## 四、报告审批

### 4.1 审核报告

**适用角色**：审核人

**入口**：报告管理 → 待审核

或从首页「待办事项」进入。

#### 审核内容

| 检查项 | 说明 |
|--------|------|
| 数据一致性 | 报告数据与原始记录一致 |
| 结论正确性 | 判定结论正确 |
| 格式规范性 | 报告格式符合要求 |
| 信息完整性 | 所有信息填写完整 |

#### 审核操作

**审核通过**：
1. 点击「通过」按钮
2. 填写审核意见
3. 报告进入待批准状态

**审核退回**：
1. 点击「退回」按钮
2. 填写退回原因
3. 报告退回给编制人修改

### 4.2 批准报告

**适用角色**：授权签字人

**入口**：报告管理 → 待批准

或从首页「待办事项」进入。

#### 批准前检查

- [ ] 确认检测项目在授权范围内
- [ ] 确认检测方法有效
- [ ] 确认数据准确
- [ ] 确认结论正确

#### 批准操作

1. 查看报告详情
2. 确认内容无误
3. 点击「批准」按钮
4. 系统自动签署电子签名
5. 报告状态变更为「已批准」

## 五、电子签名

### 5.1 签名要求

- 签名图片格式：PNG、JPG
- 文件大小：不超过2MB
- 背景：透明或白色
- 签名清晰可辨

### 5.2 上传签名

**路径**：个人中心 → 电子签名

1. 点击「上传签名」按钮
2. 选择签名图片文件
3. 系统验证格式和大小
4. 上传成功

### 5.3 签名使用

批准报告时：
- 系统自动将签名添加到报告
- 签名位置：批准人签字栏
- 签名时间：系统自动记录

### 5.4 签名验证

报告上的电子签名可验证：
- 签名人身份
- 签名时间
- 签名有效性

## 六、报告发放

### 6.1 发放登记

**适用角色**：接待员

**入口**：报告管理 → 报告发放

#### 发放方式

| 方式 | 说明 |
|------|------|
| 纸质报告 | 打印纸质报告交付 |
| 电子报告 | 发送PDF文件 |
| 双份 | 纸质+电子 |

#### 发放操作

1. 选择已批准的报告
2. 点击「发放」按钮
3. 填写发放信息：

| 字段 | 说明 |
|------|------|
| 发放方式 | 纸质/电子/双份 |
| 接收人 | 接收人姓名 |
| 接收单位 | 接收单位名称 |
| 联系电话 | 接收人电话 |
| 发放日期 | 实际发放日期 |
| 备注 | 其他说明 |

4. 确认发放

### 6.2 发放记录

所有发放记录保存在系统中，可查询：
- 发放日期
- 发放方式
- 接收人信息
- 发放人

## 九、报告作废

### 9.1 作废条件

以下情况可作废报告：
- 检测数据错误
- 样品信息错误
- 委托方要求
- 其他原因

### 9.2 作废操作

1. 进入报告详情页
2. 点击「作废」按钮
3. 填写作废原因
4. 确认作废

### 9.3 作废后处理

- 报告状态变更为「已作废」
- 原报告标记作废
- 如需重新出报告，需新建报告

## 十、常见问题

### Q1: 如何查看报告的审批进度？

**方法**：进入报告详情页，查看「审批记录」标签页，可看到完整的审批流程和当前状态。

### Q2: 报告批准后发现错误怎么办？

**处理方法**：
1. 作废原报告
2. 修改数据
3. 重新走审批流程
4. 出具新报告

### Q3: 如何查看历史报告？

**方法**：在报告列表页设置筛选条件，可查看所有历史报告。

### Q4: 报告可以修改吗？

**答案**：已批准的报告不能修改。如需修改，需作废后重新出报告。

### Q5: 电子签名具有法律效力吗？

**答案**：是的。根据《电子签名法》，可靠的电子签名与手写签名具有同等法律效力。
