# 原始记录

本文档介绍原始记录的填写、复核和管理流程。

---

## 一、功能概述

原始记录是检测过程的原始数据记录，是检测结果的真实反映。主要功能包括：

- **记录填写**：使用模板填写检测数据
- **自动计算**：系统自动计算检测结果
- **记录复核**：复核人审核记录
- **修改留痕**：记录所有修改历史

### 1.1 适用角色

| 角色 | 权限 |
|------|------|
| 检测人员 | 填写、编辑草稿记录 |
| 复核人/审核人 | 复核记录 |
| 技术负责人 | 查看所有记录 |

### 1.2 记录状态流转

```
┌─────────┐   提交    ┌───────────┐   通过    ┌─────────┐
│  草稿   │ ───────▶ │  待复核   │ ───────▶ │ 已复核  │
│ (draft) │          │(pending)  │          │(reviewed)│
└─────────┘          └───────────┘          └─────────┘
                          │                      
                          │ 退回                 
                          ▼                      
                     ┌─────────┐               
                     │ 已退回  │ ──▶ 修改后重新提交
                     │(returned)│              
                     └─────────┘               
```

### 2.1 模板类型

系统预置了常用检测类型的记录模板：

| 模板名称 | 适用检测 |
|----------|----------|
| 混凝土抗压强度 | 混凝土立方体抗压强度 |
| 混凝土抗折强度 | 混凝土抗折强度 |
| 钢筋拉伸试验 | 钢筋屈服强度、抗拉强度、伸长率 |
| 钢筋弯曲试验 | 钢筋弯曲性能 |
| 沥青针入度 | 沥青针入度 |
| 沥青软化点 | 沥青软化点 |
| 沥青延度 | 沥青延度 |
| 土工含水率 | 土壤含水率 |
| 土工密度 | 土壤密度、干密度 |
| 压实度 | 压实度检测 |

### 2.2 模板结构

每个模板包含以下部分：

1. **基本信息区**
   - 任务编号
   - 样品编号
   - 检测日期
   - 环境条件

2. **设备信息区**
   - 设备名称
   - 设备编号
   - 校准有效期

3. **检测数据区**
   - 测量值输入
   - 自动计算字段
   - 单位显示

4. **结果判定区**
   - 检测结果
   - 判定结论

## 三、记录填写

### 3.1 新建记录

**路径**：检测业务 → 原始记录 → 新建

或从检测任务详情页点击「填写记录」。

### 3.2 选择模板

1. 选择检测任务
2. 系统自动匹配模板
3. 或手动选择其他模板

### 3.3 填写基本信息

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 任务编号 | 自动 | 关联检测任务 |
| 样品编号 | 自动 | 盲样编号 |
| 检测日期 | 是 | 默认当天 |
| 环境温度 | 是 | 检测环境温度 |
| 环境湿度 | 是 | 检测环境湿度 |

### 3.4 填写检测数据

根据模板填写测量数据：

**示例：混凝土抗压强度**

| 试件编号 | 尺寸(mm) | 受压面积(mm²) | 破坏荷载(kN) | 抗压强度(MPa) |
|----------|----------|---------------|--------------|---------------|
| 1 | 150×150×150 | 22500 | 680 | 30.2 |
| 2 | 150×150×150 | 22500 | 710 | 31.6 |
| 3 | 150×150×150 | 22500 | 695 | 30.9 |

**说明**：
- 受压面积：根据尺寸自动计算
- 抗压强度：根据荷载和面积自动计算
- 平均值：系统自动计算

### 3.5 自动计算

系统支持以下自动计算：

#### 混凝土相关
- 抗压强度 = 荷载 × 1000 / 面积
- 抗折强度 = 荷载 × 跨距 / (宽 × 高²)
- 尺寸修正系数

#### 钢筋相关
- 横截面积 = π × 直径² / 4
- 屈服强度 = 屈服荷载 × 1000 / 面积
- 抗拉强度 = 最大荷载 × 1000 / 面积
- 伸长率 = (断后标距 - 原始标距) / 原始标距 × 100
- 强屈比 = 抗拉强度 / 屈服强度

#### 土工相关
- 含水率 = (湿土质量 - 干土质量) / 干土质量 × 100
- 干密度 = 湿密度 / (1 + 含水率)
- 压实度 = 干密度 / 最大干密度 × 100

### 3.6 结果判定

系统根据预设规则自动判定：

| 判定类型 | 规则示例 |
|----------|----------|
| 单值判定 | 抗压强度 ≥ 设计强度等级 |
| 平均值判定 | 平均强度 ≥ 设计强度等级的95% |
| 范围判定 | 结果在允许范围内 |

### 3.7 保存记录

- **保存草稿**：保存但不提交，可继续编辑
- **提交复核**：提交给复核人审核

## 四、记录复核

### 4.1 复核入口

**路径**：检测业务 → 原始记录 → 待复核

或从首页「待办事项」进入。

### 4.2 复核内容

复核人需检查以下内容：

| 检查项 | 说明 |
|--------|------|
| 数据完整性 | 所有必填项是否填写 |
| 数据合理性 | 数值是否在合理范围内 |
| 计算正确性 | 自动计算结果是否正确 |
| 判定准确性 | 判定结论是否符合标准 |
| 格式规范性 | 修约、单位是否正确 |
| 环境条件 | 是否满足检测要求 |

### 4.3 复核操作

**复核通过**：
1. 点击「通过」按钮
2. 填写复核意见（可选）
3. 记录状态变更为「已复核」

**复核退回**：
1. 点击「退回」按钮
2. 填写退回原因（必填）
3. 记录状态变更为「已退回」
4. 检测人员收到通知

## 五、记录修改

### 5.1 草稿修改

草稿状态的记录可以自由修改，不留痕迹。

### 5.2 已提交记录修改

已提交的记录如需修改：

1. 联系复核人退回
2. 退回后可修改
3. 修改会自动记录痕迹

### 5.3 修改痕迹记录

系统自动记录以下信息：

| 字段 | 说明 |
|------|------|
| 修改字段 | 字段路径 |
| 原值 | 修改前的值 |
| 新值 | 修改后的值 |
| 修改人 | 操作用户 |
| 修改时间 | 操作时间 |
| 修改原因 | 用户填写 |

### 5.4 查看修改痕迹

在记录详情页点击「修改痕迹」标签页，可查看所有修改历史。

## 六、数据完整性校验

### 6.1 哈希校验

系统对原始记录数据进行哈希校验：

- 记录提交时计算数据哈希值
- 哈希值存储在数据库中
- 每次查看时校验哈希值
- 如哈希值不匹配，提示数据可能被篡改

### 6.2 数据锁定

复核通过的记录会被锁定：
- 不能直接修改
- 如需修改需走变更流程
- 变更后重新复核

## 九、常见问题

### Q1: 记录提交后发现错误怎么办？

**解决方法**：联系复核人退回记录，修改后重新提交。

### Q2: 如何查看修改历史？

**方法**：进入记录详情页，点击「修改痕迹」标签页。

### Q3: 自动计算结果不正确怎么办？

**排查步骤**：
1. 检查输入数据是否正确
2. 检查单位是否正确
3. 检查公式参数设置
4. 如仍有问题，联系技术负责人

### Q4: 环境条件超出范围怎么办？

**处理方法**：
- 如超出允许范围，系统会提示警告
- 需在备注中说明情况
- 必要时重新检测

### Q5: 记录可以删除吗？

**答案**：已提交的记录不能删除，只能作废。草稿状态的记录可以删除。
