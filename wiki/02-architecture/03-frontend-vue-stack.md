# 前端技术栈（Vue）

- **框架**：Vue 3、Composition API、TypeScript。
- **UI**：Element Plus。
- **路由**：`vue-router`，路由元信息配合 **权限** 过滤菜单。
- **HTTP**：封装 `request.ts`，需兼容 **业务信封** `code` 与数字/字符串（见 [前端信封](../05-api-and-integration/04-frontend-api-envelope.md)）。

调试：[调试手册](../03-development/05-debugging-playbook.md)
