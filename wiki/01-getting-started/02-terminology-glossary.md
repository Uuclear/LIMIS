# 术语与缩写表

| 术语 | 说明 |
|------|------|
| LIMIS | 实验室信息管理系统（本系统） |
| 委托 / 委托单 | 客户发起的检测委托业务单元 |
| 样品 | 待检或已检的实物对象及其元数据 |
| 检测任务 | 实验室执行检测的工作单元 |
| 原始记录 | 按模板记录的实验过程与读数 |
| 检测结果 | 结构化检验结果，可能用于报告 |
| JWT | JSON Web Token，用于 API 访问令牌 |
| `sv` / session_version | 会话版本号，用于使旧 Token 失效 |
| DRF | Django REST Framework |
| 模块权限 | `lims_module` + `view/create/edit/delete` 级控制 |

更多业务词参见 [用户指南](../06-user-guide/01-entrustment-and-projects.md) 与 [业务流程 FAQ](../07-faq/02-business-process-faq.md)。
