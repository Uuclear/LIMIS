# 界面导航与菜单地图

侧边栏按业务域分组，实际可见项受 **路由权限** 过滤。以下为典型结构（与前端 `Sidebar` 一致，版本迭代时可能微调）。

| 一级 | 二级示例 |
|------|----------|
| 首页 | `/dashboard` |
| 业务管理 | 工程项目、委托管理、样品管理 |
| 检测管理 | 检测任务、原始记录、检测结果 |
| 报告管理 | 报告列表 |
| 资源管理 | 仪器设备、人员管理、耗材管理 |
| 监控管理 | 环境监控 |
| 质量体系 | 检测基础配置、标准规范、项目参数库、原始记录模板、报告模板、资质管理、内部审核、管理评审、不符合项 |
| 系统管理 | 用户管理、角色管理、操作日志 |

## 权限与菜单

无权限的菜单项 **不显示**；直接访问 URL 可能返回 **403**。详见 [授权模型](../08-security-compliance/01-authorization-model.md)。
