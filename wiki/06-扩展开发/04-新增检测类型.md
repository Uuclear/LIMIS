# 新增检测类型

本文档详细介绍如何在 LIMIS 系统中新增检测类型，包括检测类别、检测方法、检测参数、原始记录模板、计算公式和判定规则的配置。

## 1. 概述

### 1.1 检测类型层级结构

LIMIS 系统的检测类型采用层级结构：

```
检测类别 (TestCategory)
    └── 检测方法 (TestMethod)
            └── 检测参数 (TestParameter)
                    └── 判定规则 (JudgmentRule)
```

### 1.2 新增检测类型流程

```
1. 创建检测类别
2. 创建检测方法
3. 配置检测参数
4. 创建原始记录模板
5. 配置计算公式
6. 配置判定规则
7. 测试验证
```

## 2. 添加检测类别

### 2.1 检测类别模型

```python
# apps/testing/models.py

class TestCategory(BaseModel):
    """检测类别"""
    
    name = models.CharField(
        max_length=100,
        verbose_name='类别名称'
    )
    code = models.CharField(
        max_length=50,
        unique=True,
        verbose_name='类别编码'
    )
    parent = models.ForeignKey(
        'self',
        on_delete=models.CASCADE,
        null=True,
        blank=True,
        related_name='children',
        verbose_name='父级类别'
    )
    sort_order = models.IntegerField(
        default=0,
        verbose_name='排序'
    )
    description = models.TextField(
        blank=True,
        verbose_name='类别描述'
    )
    
    class Meta:
        verbose_name = '检测类别'
        verbose_name_plural = verbose_name
        ordering = ['sort_order', 'code']
```

### 2.2 通过管理后台添加

**步骤：**

1. 登录管理后台：`http://localhost:8000/admin/`
2. 进入"检测管理" → "检测类别"
3. 点击"添加检测类别"
4. 填写表单：
   - 类别名称：如"建筑材料"
   - 类别编码：如"JZCL"
   - 父级类别：选择或留空（顶级类别）
   - 排序：设置显示顺序
   - 类别描述：填写类别说明
5. 点击"保存"

### 2.3 通过 API 添加

**请求示例：**

```bash
curl -X POST "http://localhost:8000/api/v1/testing/categories/" \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \

```

### 2.5 类别编码规范

| 规范 | 说明 | 示例 |
|------|------|------|
| 顶级类别 | 4位大写字母 | JZCL（建筑材料） |
| 二级类别 | 顶级编码-3位缩写 | JZCL-HNT（混凝土） |
| 三级类别 | 二级编码-3位缩写 | JZCL-HNT-KY（抗压） |

## 3. 添加检测方法

### 3.1 检测方法模型

```python
# apps/testing/models.py

class TestMethod(BaseModel):
    """检测方法"""
    
    name = models.CharField(
        max_length=200,
        verbose_name='方法名称'
    )
    standard_no = models.CharField(
        max_length=50,
        verbose_name='标准编号'
    )
    standard_name = models.CharField(
        max_length=200,
        verbose_name='标准名称'
    )
    category = models.ForeignKey(
        TestCategory,
        on_delete=models.PROTECT,
        related_name='methods',
        verbose_name='检测类别'
    )
    description = models.TextField(
        blank=True,
        verbose_name='方法描述'
    )
    principle = models.TextField(
        blank=True,
        verbose_name='检测原理'
    )
    equipment_requirements = models.TextField(
        blank=True,
        verbose_name='设备要求'
    )
    environment_requirements = models.TextField(
        blank=True,
        verbose_name='环境要求'
    )
    sample_requirements = models.TextField(
        blank=True,
        verbose_name='样品要求'
    )
    procedure = models.TextField(
        blank=True,
        verbose_name='检测步骤'
    )
    is_active = models.BooleanField(
        default=True,
        verbose_name='是否启用'
    )
    
    class Meta:
        verbose_name = '检测方法'
        verbose_name_plural = verbose_name
        ordering = ['standard_no']
```

### 3.2 通过管理后台添加

**步骤：**

1. 进入"检测管理" → "检测方法"
2. 点击"添加检测方法"
3. 填写表单：
   - 方法名称：如"混凝土抗压强度试验"
   - 标准编号：如"GB/T 50081-2019"
   - 标准名称：如"混凝土物理力学性能试验方法标准"
   - 检测类别：选择对应的类别
   - 方法描述：填写方法概述
   - 检测原理：填写检测原理说明
   - 设备要求：填写所需设备
   - 环境要求：填写环境条件要求
   - 样品要求：填写样品制备要求
   - 检测步骤：填写详细操作步骤
4. 点击"保存"

## 4. 添加检测参数

### 4.1 检测参数模型

```python
# apps/testing/models.py

class TestParameter(BaseModel):
    """检测参数"""
    
    method = models.ForeignKey(
        TestMethod,
        on_delete=models.CASCADE,
        related_name='parameters',
        verbose_name='检测方法'
    )
    name = models.CharField(
        max_length=100,
        verbose_name='参数名称'
    )
    code = models.CharField(
        max_length=50,
        verbose_name='参数编码'
    )
    unit = models.CharField(
        max_length=20,
        blank=True,
        verbose_name='单位'
    )
    precision = models.IntegerField(
        default=2,
        verbose_name='精度(小数位数)'
    )
    rounding_rule = models.CharField(
        max_length=20,
        default='half_up',
        verbose_name='修约规则'
    )
    min_value = models.DecimalField(
        max_digits=12,
        decimal_places=4,
        null=True,
        blank=True,
        verbose_name='最小值'
    )
    max_value = models.DecimalField(
        max_digits=12,
        decimal_places=4,
        null=True,
        blank=True,
        verbose_name='最大值'
    )
    is_required = models.BooleanField(
        default=True,
        verbose_name='是否必检'
    )
    formula = models.TextField(
        blank=True,
        verbose_name='计算公式'
    )
    input_type = models.CharField(
        max_length=20,
        default='number',
        verbose_name='输入类型'
    )
    default_value = models.CharField(
        max_length=100,
        blank=True,
        verbose_name='默认值'
    )
    
    class Meta:
        verbose_name = '检测参数'
        verbose_name_plural = verbose_name
        ordering = ['method', 'code']
        unique_together = [['method', 'code']]
```

### 4.2 通过管理后台添加

**步骤：**

1. 进入"检测管理" → "检测参数"
2. 点击"添加检测参数"
3. 填写表单：
   - 检测方法：选择对应方法
   - 参数名称：如"抗压强度"
   - 参数编码：如"KY-QD"
   - 单位：如"MPa"
   - 精度：如1（保留1位小数）
   - 修约规则：选择修约方式
   - 最小值/最大值：设置范围
   - 是否必检：勾选
   - 计算公式：填写计算公式
   - 输入类型：选择 number/text/select 等
4. 点击"保存"

| 输入类型 | 说明 | 示例 |

### 4.6 修约规则说明

| 修约规则 | 说明 | 示例 |
|----------|------|------|
| half_up | 四舍五入 | 23.45 → 23.5 |
| half_down | 五舍六入 | 23.45 → 23.4 |
| up | 向上取整 | 23.41 → 23.5 |
| down | 向下取整 | 23.49 → 23.4 |
| ceiling | 正数向上，负数向下 | -23.45 → -23.4 |
| floor | 正数向下，负数向上 | -23.45 → -23.5 |
| truncate | 截断 | 23.45 → 23.4 |

### 5.1 原始记录模板模型

# apps/testing/models.py

class RecordTemplate(BaseModel):
    """原始记录模板"""
    
    name = models.CharField(
        max_length=200,
        verbose_name='模板名称'
    )
    code = models.CharField(
        max_length=50,
        unique=True,
        verbose_name='模板编码'
    )
    test_method = models.ForeignKey(
        TestMethod,
        on_delete=models.PROTECT,
        related_name='record_templates',
        verbose_name='检测方法'
    )
    test_parameter = models.ForeignKey(
        TestParameter,
        on_delete=models.PROTECT,
        null=True,
        blank=True,
        related_name='record_templates',
        verbose_name='检测参数'
    )
    version = models.CharField(
        max_length=20,
        default='1.0',
        verbose_name='版本号'
    )
    schema = models.JSONField(
        verbose_name='模板结构'
    )
    layout = models.JSONField(
        blank=True,
        null=True,
        verbose_name='布局配置'
    )
    is_active = models.BooleanField(
        default=True,
        verbose_name='是否启用'
    )
    effective_date = models.DateField(
        verbose_name='生效日期'
    )
    expiry_date = models.DateField(
        null=True,
        blank=True,
        verbose_name='失效日期'
    )
    
    class Meta:

### 6.1 公式引擎

系统内置公式计算引擎，支持多种计算表达式：

```python
# core/utils/formula.py

import re
import math
from decimal import Decimal, ROUND_HALF_UP

class FormulaEngine:
    """公式计算引擎"""
    
    SUPPORTED_FUNCTIONS = {
        'abs': abs,
        'round': round,
        'floor': math.floor,
        'ceil': math.ceil,
        'sqrt': math.sqrt,
        'pow': pow,
        'log': math.log,
        'log10': math.log10,
        'exp': math.exp,
        'sin': math.sin,
        'cos': math.cos,
        'tan': math.tan,
        'avg': lambda *args: sum(args) / len(args) if args else 0,
        'sum': sum,
        'min': min,
        'max': max,
        'std': lambda *args: math.sqrt(sum((x - sum(args)/len(args))**2 for x in args) / len(args)) if len(args) > 1 else 0,
        'count': len,
        'if': lambda cond, true_val, false_val: true_val if cond else false_val,
    }
    
    def evaluate(self, formula: str, context: dict) -> Decimal:
        """
        计算公式
        
        :param formula: 公式表达式
        :param context: 变量上下文
        :return: 计算结果
        """
        # 替换变量
        expression = self._replace_variables(formula, context)
        
        # 安全计算
        try:
            result = self._safe_eval(expression)
            return Decimal(str(result))
        except Exception as e:
            raise ValueError(f'公式计算错误: {e}')
    
    def _replace_variables(self, formula: str, context: dict) -> str:
        """替换变量为实际值"""
        for var, value in context.items():
            formula = formula.replace(var, str(value))
        return formula
    
    def _safe_eval(self, expression: str) -> float:
        """安全计算表达式"""
        # 只允许数字、运算符、函数名和括号
        allowed = r'^[\d\s\+\-\*\/\(\)\.\,\w]+$'
        if not re.match(allowed, expression):
            raise ValueError('表达式包含非法字符')
        
        # 构建安全的计算环境
        safe_dict = {k: v for k, v in self.SUPPORTED_FUNCTIONS.items()}
        safe_dict['__builtins__'] = {}
        
        return eval(expression, safe_dict)
```

### 7.4 判定引擎

```python
# apps/testing/judgment.py

from decimal import Decimal
from typing import Optional
from .models import JudgmentRule, TestResult

class JudgmentEngine:
    """判定引擎"""
    
    def judge(self, result: TestResult) -> str:
        """
        对检测结果进行判定
        
        :param result: 检测结果对象
        :return: 判定结果
        """
        rules = JudgmentRule.objects.filter(
            test_parameter=result.parameter,
            is_deleted=False,
        ).order_by('-priority')
        
        value = Decimal(str(result.raw_value))
        
        for rule in rules:
            if self._match_rule(value, rule):
                return rule.judgment_result
        
        return '待判定'
    
    def _match_rule(self, value: Decimal, rule: JudgmentRule) -> bool:
        """判断值是否匹配规则"""
        comparison = rule.comparison_type
        
        if comparison == 'range':
            min_v = rule.min_value
            max_v = rule.max_value
            if min_v is not None and max_v is not None:
                return min_v <= value <= max_v
        
        elif comparison == 'greater':
            return value > rule.min_value if rule.min_value else False
        
        elif comparison == 'greater_equal':
            return value >= rule.min_value if rule.min_value else False
        
        elif comparison == 'less':
            return value < rule.max_value if rule.max_value else False
        
        elif comparison == 'less_equal':
            return value <= rule.max_value if rule.max_value else False
        
        elif comparison == 'equal':
            return value == rule.min_value
        
        elif comparison == 'not_equal':
            return value != rule.min_value
        
        elif comparison == 'in_list':
            allowed = rule.standard_ref.split(',')
            return str(value) in allowed
        
        elif comparison == 'not_in_list':
            disallowed = rule.standard_ref.split(',')
            return str(value) not in disallowed
        
        return False
```
