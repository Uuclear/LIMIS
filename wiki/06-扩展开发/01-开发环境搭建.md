# 开发环境搭建

本文档详细介绍 LIMIS 实验室信息管理系统的开发环境配置步骤。

## 1. 系统要求

### 1.1 操作系统

- **推荐**: Ubuntu 22.04 LTS / CentOS 8+ / macOS 12+
- **支持**: Windows 10/11 (WSL2 推荐)

### 1.2 硬件要求

| 资源 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 2核 | 4核+ |
| 内存 | 4GB | 8GB+ |
| 磁盘 | 20GB | 50GB+ SSD |

## 2. Python 环境配置

### 2.1 安装 Python

系统使用 Python 3.11+ 版本。

**Ubuntu/Debian:**

```bash
# 添加 deadsnakes PPA (Ubuntu)
sudo apt update
sudo apt install -y software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update

# 安装 Python 3.11
sudo apt install -y python3.11 python3.11-venv python3.11-dev

# 安装 pip
curl -sS https://bootstrap.pypa.io/get-pip.py | sudo python3.11
```

**CentOS/RHEL:**

```bash
# 安装依赖
sudo dnf install -y gcc openssl-devel bzip2-devel libffi-devel zlib-devel

# 编译安装 Python 3.11
cd /tmp
wget https://www.python.org/ftp/python/3.11.9/Python-3.11.9.tgz
tar xzf Python-3.11.9.tgz
cd Python-3.11.9
./configure --enable-optimizations
sudo make altinstall

# 创建软链接
sudo ln -sf /usr/local/bin/python3.11 /usr/bin/python3
sudo ln -sf /usr/local/bin/pip3.11 /usr/bin/pip3
```

**macOS:**

```bash
# 使用 Homebrew
brew install python@3.11

# 设置路径
echo 'export PATH="/usr/local/opt/python@3.11/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### 2.2 创建虚拟环境

```bash
# 进入项目目录
cd /opt/limis/backend

# 创建虚拟环境
python3.11 -m venv venv

# 激活虚拟环境
# Linux/macOS
source venv/bin/activate

# Windows
.\venv\Scripts\activate
```

### 2.3 安装 Python 依赖

```bash
# 确保在虚拟环境中
pip install --upgrade pip setuptools wheel

# 安装项目依赖
pip install -r requirements.txt

# 安装开发依赖
```


## 4. PostgreSQL 安装

### 4.1 安装 PostgreSQL

**Ubuntu/Debian:**

```bash
# 安装 PostgreSQL 15
sudo apt install -y postgresql-15 postgresql-contrib-15

# 启动服务
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

**CentOS/RHEL:**

```bash
# 添加 PostgreSQL 仓库
sudo dnf install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-9-x86_64/pgdg-redhat-repo-latest.noarch.rpm

# 安装 PostgreSQL 15
sudo dnf install -y postgresql15-server postgresql15

# 初始化数据库
sudo /usr/pgsql-15/bin/postgresql-15-setup initdb

# 启动服务
sudo systemctl start postgresql-15
sudo systemctl enable postgresql-15
```

**macOS:**

```bash
# 使用 Homebrew
brew install postgresql@15

# 启动服务
brew services start postgresql@15
```

### 4.2 创建数据库和用户

```bash
# 切换到 postgres 用户
sudo -u postgres psql

# 在 psql 中执行
CREATE USER limis WITH PASSWORD 'your_secure_password';
CREATE DATABASE limis OWNER limis ENCODING 'UTF8';
GRANT ALL PRIVILEGES ON DATABASE limis TO limis;

# 退出 psql
\q
```

### 4.3 配置远程访问 (开发环境)

编辑 `postgresql.conf`:

```bash
sudo vim /etc/postgresql/15/main/postgresql.conf
```

```ini
# 修改监听地址
listen_addresses = '*'
port = 5432
```

编辑 `pg_hba.conf`:

```bash
sudo vim /etc/postgresql/15/main/pg_hba.conf
```

```ini
# 添加允许的连接
host    all             all             0.0.0.0/0               md5
```

重启 PostgreSQL:

```bash
sudo systemctl restart postgresql
```

### 5.1 安装 Redis

**Ubuntu/Debian:**

```bash
# 安装 Redis
sudo apt install -y redis-server

# 启动服务
sudo systemctl start redis-server
sudo systemctl enable redis-server
```

**CentOS/RHEL:**

```bash
# 安装 Redis
sudo dnf install -y redis

# 启动服务
sudo systemctl start redis
sudo systemctl enable redis
```

**macOS:**

```bash
# 使用 Homebrew
brew install redis

# 启动服务
brew services start redis
```

### 5.2 配置 Redis

编辑配置文件:

```bash
sudo vim /etc/redis/redis.conf
```

```ini
# 绑定地址 (开发环境)
bind 0.0.0.0

# 设置密码 (生产环境必须)
requirepass your_redis_password

# 最大内存
maxmemory 256mb
maxmemory-policy allkeys-lru
```

重启 Redis:

```bash
sudo systemctl restart redis-server
```

### 5.3 验证 Redis 连接

```bash
redis-cli ping
# 应返回 PONG
```

## 6. 项目克隆和依赖安装


### 6.2 配置环境变量

复制环境变量模板:

```bash
cp .env.example .env
```

编辑 `.env` 文件:

```bash
# Django 配置
DJANGO_SETTINGS_MODULE=limis.settings.dev
SECRET_KEY=your-secret-key-here-change-in-production
DEBUG=True

# 数据库配置
DB_NAME=limis
DB_USER=limis
DB_PASSWORD=your_secure_password
DB_HOST=localhost
DB_PORT=5432

# Redis 配置
REDIS_URL=redis://localhost:6379/0

# MinIO 配置
MINIO_ENDPOINT=localhost:9000
MINIO_ACCESS_KEY=minioadmin
MINIO_SECRET_KEY=minioadmin
MINIO_BUCKET=limis

# JWT 配置
JWT_ACCESS_TOKEN_LIFETIME=120  # 分钟
JWT_REFRESH_TOKEN_LIFETIME=10080  # 分钟 (7天)
```

### 6.3 安装后端依赖

```bash
cd /opt/limis/backend

python3 -m venv venv
```


## 7. 开发服务器启动

### 7.1 启动后端服务

**方式一: 直接启动 Django**

```bash
cd /opt/limis/backend
source venv/bin/activate

# 启动开发服务器
python manage.py runserver 0.0.0.0:8000
```

**方式二: 使用 Django 扩展**

```bash
# 使用 django-extensions 的 runserver_plus (支持调试工具栏)
python manage.py runserver_plus 0.0.0.0:8000
```

**方式三: 启动 Celery Worker (异步任务)**

```bash
# 启动 Celery Worker
celery -A limis worker -l info

# 启动 Celery Beat (定时任务)
celery -A limis beat -l info
```

### 7.2 启动前端服务

```bash
cd /opt/limis/frontend

# 开发模式启动
npm run dev

# 或使用 pnpm
pnpm dev
```

前端开发服务器默认运行在 `http://localhost:5173`。

### 7.3 使用 Docker Compose (推荐)

项目提供了 Docker Compose 配置，可一键启动所有服务:

```bash
cd /opt/limis

# 启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f backend
docker-compose logs -f frontend
```

**docker-compose.yml 示例:**

```yaml
version: '3.8'

services:
  db:
    image: postgres:15
    environment:
      POSTGRES_DB: limis
      POSTGRES_USER: limis
      POSTGRES_PASSWORD: limis_password
    volumes:
      - postgres_data:/var/lib/postgresql/data
    ports:
      - "5432:5432"

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

  minio:
    image: minio/minio
    command: server /data --console-address ":9001"
    environment:
      MINIO_ROOT_USER: minioadmin
      MINIO_ROOT_PASSWORD: minioadmin
    volumes:
      - minio_data:/data
    ports:
      - "9000:9000"
      - "9001:9001"

  backend:
    build: ./backend
    command: python manage.py runserver 0.0.0.0:8000
    volumes:
      - ./backend:/app
    ports:
      - "8000:8000"
    environment:
      - DJANGO_SETTINGS_MODULE=limis.settings.dev
      - DB_HOST=db
      - DB_NAME=limis
      - DB_USER=limis
      - DB_PASSWORD=limis_password
      - REDIS_URL=redis://redis:6379/0
    depends_on:
      - db
      - redis

  frontend:
    build: ./frontend
    command: npm run dev -- --host
    volumes:
      - ./frontend:/app
      - /app/node_modules
    ports:
      - "5173:5173"
    depends_on:
      - backend

volumes:
  postgres_data:
  minio_data:
```

## 9. 常见问题

### 9.1 数据库连接失败

```bash
# 检查 PostgreSQL 服务状态
sudo systemctl status postgresql

# 检查连接
psql -h localhost -U limis -d limis

# 检查防火墙
sudo ufw allow 5432/tcp
```

### 9.2 Redis 连接失败

```bash
# 检查 Redis 服务状态
sudo systemctl status redis-server

# 测试连接
redis-cli ping
```

### 9.3 前端依赖安装失败

```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
```

### 9.4 Python 依赖冲突

```bash
# 重建虚拟环境
rm -rf venv
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
