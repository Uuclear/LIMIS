# 快速开始

## 一、系统登录

### 1.1 访问系统

在浏览器地址栏输入系统地址，例如：
- 生产环境：`https://limis.example.com`
- 测试环境：`https://limis-test.example.com`

### 1.2 登录界面

![登录界面](./images/login.png)

| 字段 | 说明 |
|------|------|
| 用户名 | 系统分配的用户名 |
| 密码 | 初始密码由管理员提供，首次登录需修改 |

### 1.3 登录步骤

1. 输入用户名和密码
2. 点击「登录」按钮
3. 首次登录系统会提示修改密码
4. 登录成功后跳转到首页仪表盘

### 1.4 忘记密码

如忘记密码，请联系系统管理员重置。管理员重置后，您将收到临时密码，登录后需立即修改。

## 三、基本操作

### 3.1 列表页面操作

大多数模块的列表页面具有统一的操作方式：

```
┌─────────────────────────────────────────────────────────────────┐
│  页面标题                                    [新增] [导出]      │
├─────────────────────────────────────────────────────────────────┤
│  筛选条件：[状态 ▼] [日期范围] [关键词搜索] [查询] [重置]        │
├─────────────────────────────────────────────────────────────────┤
│  ☑ | 编号 | 名称 | 状态 | 创建时间 | 操作                       │
│  ───┼──────┼──────┼──────┼──────────┼───────────────────        │
│  ☐ | P001 | XX项目| 进行中| 2026-03-01| [查看] [编辑] [删除]    │
│  ☐ | P002 | YY项目| 已完成| 2026-03-02| [查看] [编辑] [删除]    │
│  ...                                                            │
├─────────────────────────────────────────────────────────────────┤
│  共 100 条  每页 20 条  < 1 2 3 4 5 ... >                       │
└─────────────────────────────────────────────────────────────────┘
```

#### 筛选功能

- **下拉筛选**：按状态、类型等字段筛选
- **日期筛选**：选择日期范围
- **关键词搜索**：输入关键词模糊搜索
- **组合筛选**：多个条件同时生效

#### 批量操作

1. 勾选需要操作的记录（可全选）
2. 点击批量操作按钮（如批量删除、批量导出）
3. 确认操作

#### 分页

- 默认每页显示20条
- 可选择每页显示数量（20/50/100）
- 显示总记录数和当前页码

#### 表单验证

- **必填字段**：标有红色星号 * 的字段必须填写
- **格式验证**：系统自动验证邮箱、手机号、日期等格式
- **逻辑验证**：如结束日期不能早于开始日期

#### 保存操作

- **保存**：保存后返回列表页
- **保存并继续**：保存后留在当前页面，可继续编辑
- **取消**：放弃修改，返回上一页

## 五、消息通知

### 5.1 通知类型

系统会在以下情况发送通知：

| 通知类型 | 触发条件 | 接收人 |
|----------|----------|--------|
| 委托待评审 | 委托单提交后 | 技术负责人 |
| 任务已分配 | 任务分配后 | 检测人员 |
| 记录待复核 | 记录提交后 | 复核人 |
| 报告待审核 | 报告提交后 | 审核人 |
| 报告待批准 | 报告审核通过后 | 授权签字人 |
| 设备即将到期 | 校准到期前30天 | 设备管理员、技术负责人 |
| 样品龄期到期 | 龄期到期前3天 | 检测人员 |

### 5.2 查看通知

1. 点击顶部导航栏的通知图标
2. 查看未读通知列表
3. 点击通知跳转到相关页面
4. 可标记已读或全部已读

## 六、退出登录

### 6.1 安全退出

1. 点击右上角用户头像
2. 选择「退出登录」
3. 系统清除登录状态，跳转到登录页

### 6.2 自动退出

- 系统空闲超过30分钟自动退出
- 需重新登录才能继续使用

## 七、常见问题

### Q1: 登录时提示"用户名或密码错误"

**解决方法**：
1. 检查用户名是否正确（区分大小写）
2. 检查密码是否正确（区分大小写）
3. 如多次尝试失败，联系管理员重置密码

### Q2: 页面加载缓慢

**解决方法**：
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器

### Q3: 无法看到某些菜单

**原因**：权限不足

**解决方法**：联系管理员分配相应角色权限
