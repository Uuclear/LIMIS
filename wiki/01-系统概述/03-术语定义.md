# 术语定义

本文档定义了系统中使用的业务术语和技术术语，帮助用户理解系统功能。

---

## 一、业务术语

### 1.1 组织架构相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 工程项目 | Project | 指具体的建设工程，如浦东国际机场四期扩建工程 |
| 参建单位 | Organization | 参与工程建设的单位，包括建设、施工、监理、设计、检测等 |
| 建设单位 | Owner | 工程项目的投资方和业主 |
| 施工单位 | Contractor | 承担工程施工的企业 |
| 监理单位 | Supervisor | 对工程质量进行监督管理的单位 |
| 检测单位 | Testing Lab | 具有资质的检测机构 |
| 见证人 | Witness | 由监理单位派出，见证取样过程的人员 |
| 分部分项工程 | Sub-project | 工程项目的分解结构，如基础工程、主体结构等 |

### 1.2 委托相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 委托单 | Commission | 委托方提交的检测申请单据 |
| 委托项目 | Commission Item | 委托单中的具体检测项目明细 |
| 合同评审 | Contract Review | 对委托单进行技术能力和资源评估的过程 |
| 检测对象 | Test Object | 被检测的材料或构件，如混凝土、钢筋、沥青等 |
| 检测项目 | Test Parameter | 具体的检测参数，如抗压强度、屈服强度等 |
| 检测标准 | Test Standard | 检测依据的国家或行业标准 |
| 规格型号 | Specification | 材料的规格，如C30混凝土、HRB400钢筋等 |

### 1.3 样品相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 样品 | Sample | 用于检测的代表性材料或构件 |
| 样品编号 | Sample No. | 样品的唯一标识编号 |
| 盲样编号 | Blind No. | 隐藏委托方信息的样品编号，用于保证公正性 |
| 取样日期 | Sampling Date | 样品取样的日期 |
| 收样日期 | Received Date | 样品送达实验室的日期 |
| 样品状态 | Sample Status | 样品在流转过程中的状态，如待检、检测中、已检等 |
| 留样 | Retention | 检测完成后保留的样品，用于复检 |
| 留样期限 | Retention Period | 留样保存的时间期限 |
| 样品处置 | Sample Disposal | 留样到期后的处理方式，如退还、销毁、丢弃 |
| 样品组 | Sample Group | 同批次、同类型的样品集合 |

### 1.4 检测相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 检测任务 | Test Task | 分配给检测人员的具体检测工作 |
| 检测方法 | Test Method | 依据标准规定的检测操作方法 |
| 检测参数 | Test Parameter | 检测的具体指标，如强度、密度、含水率等 |
| 检测类别 | Test Category | 检测的分类，如混凝土检测、钢筋检测等 |
| 原始记录 | Original Record | 检测过程中记录的原始数据和观察结果 |
| 检测结果 | Test Result | 检测参数的最终数值和判定结论 |
| 判定规则 | Judgment Rule | 判断检测结果是否合格的标准 |
| 修约值 | Rounded Value | 按规定修约规则处理后的数值 |
| 龄期 | Age | 混凝土等材料从成型到检测的时间间隔，如7天、28天 |
| 平行样 | Parallel Sample | 同一样品分成多份进行平行检测 |

### 1.5 报告相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 检测报告 | Test Report | 记录检测结果和结论的正式文件 |
| 报告编号 | Report No. | 报告的唯一标识编号 |
| 报告状态 | Report Status | 报告在审批流程中的状态 |
| 编制人 | Compiler | 报告的编写人员 |
| 审核人 | Reviewer | 对报告进行技术审核的人员 |
| 批准人 | Approver | 对报告进行最终批准的人员 |
| 授权签字人 | Authorized Signatory | 经授权可签署报告的人员 |
| 电子签名 | Electronic Signature | 以电子形式存在的签名，具有法律效力 |
| 报告发放 | Report Distribution | 将报告交付给委托方的过程 |
| 报告归档 | Report Archive | 报告发放后的存档管理 |
| 报告作废 | Report Void | 报告因错误或其他原因被废止 |

### 1.6 设备相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 仪器设备 | Equipment | 用于检测的仪器和设备 |
| 设备编号 | Equipment No. | 设备的唯一标识编号 |
| 设备分类 | Equipment Category | 按管理要求分为A类（强检）、B类（非强检）、C类（辅助） |
| 检定 | Verification | 由法定计量机构进行的强制性计量检定 |
| 校准 | Calibration | 确定测量设备示值误差的操作 |
| 期间核查 | Period Check | 在两次校准之间对设备进行的核查 |
| 校准证书 | Calibration Certificate | 校准机构出具的证明文件 |
| 校准有效期 | Calibration Validity | 校准结果的有效期限 |
| 设备使用记录 | Usage Log | 设备使用情况的历史记录 |
| 维护保养 | Maintenance | 设备的日常维护和定期保养 |

### 1.7 人员相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 人员档案 | Staff Profile | 检测人员的个人信息和工作经历 |
| 上岗证书 | Certificate | 人员从事特定检测工作的资格证书 |
| 检测授权 | Authorization | 批准人员从事特定检测项目的正式授权 |
| 能力评价 | Competency Evaluation | 对人员技术能力的定期评价 |
| 培训记录 | Training Record | 人员参加培训的历史记录 |
| 签名样本 | Signature Sample | 人员签名的电子图像，用于报告签署 |

### 1.8 质量相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 内部审核 | Internal Audit | 实验室自行组织的体系审核 |
| 审核发现 | Audit Finding | 审核过程中发现的问题 |
| 不符合项 | Non-conformity | 不满足标准或程序要求的事项 |
| 纠正措施 | Corrective Action | 消除不符合原因的措施 |
| 预防措施 | Preventive Action | 防止潜在不符合发生的措施 |
| 管理评审 | Management Review | 最高管理者对体系的定期评审 |
| 能力验证 | Proficiency Test | 通过比对验证实验室检测能力 |
| 质量监督 | Quality Supervision | 对检测过程的质量监控 |
| 投诉 | Complaint | 客户对服务不满意的表达 |
| 持续改进 | Continuous Improvement | 不断提高体系有效性的过程 |

### 1.9 环境相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 监控点位 | Monitoring Point | 安装环境监测设备的位置 |
| 温度阈值 | Temperature Threshold | 温度的上下限值 |
| 湿度阈值 | Humidity Threshold | 湿度的上下限值 |
| 环境报警 | Environment Alarm | 环境参数超限时的警报 |

### 1.10 耗材相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 耗材 | Consumable | 检测过程中消耗的材料 |
| 供应商 | Supplier | 耗材的供应单位 |
| 安全库存 | Safety Stock | 最低库存警戒线 |
| 入库 | Stock In | 耗材入库登记 |
| 出库 | Stock Out | 耗材领用出库 |
| 库存预警 | Stock Alert | 库存低于安全库存时的提醒 |

## 二、系统术语

### 2.1 用户权限相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 用户 | User | 系统的使用者 |
| 角色 | Role | 一组权限的集合 |
| 权限 | Permission | 执行特定操作的授权 |
| RBAC | Role-Based Access Control | 基于角色的访问控制 |
| 登录 | Login | 用户身份认证过程 |
| 登出 | Logout | 退出系统，清除登录状态 |
| Token | Token | 用户登录后的身份凭证 |
| JWT | JSON Web Token | 一种Token格式标准 |

### 2.2 数据操作相关

| 术语 | 英文 | 定义 |
|------|------|------|
| CRUD | Create/Read/Update/Delete | 增删改查四种基本操作 |
| 列表 | List | 数据的表格展示页面 |
| 详情 | Detail | 单条数据的详细信息页面 |
| 表单 | Form | 数据录入或编辑的界面 |
| 筛选 | Filter | 按条件过滤数据 |
| 排序 | Sort | 按字段排列数据顺序 |
| 分页 | Pagination | 将数据分页显示 |
| 导出 | Export | 将数据导出为文件 |
| 导入 | Import | 从文件批量导入数据 |
| 软删除 | Soft Delete | 标记删除而非物理删除，数据可恢复 |

### 2.3 状态机相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 状态 | Status | 对象在生命周期中的当前阶段 |
| 状态流转 | Status Transition | 从一个状态变更为另一个状态 |
| 状态机 | State Machine | 定义合法状态流转的规则 |
| 终态 | Final State | 不可再变更的状态 |
| 回退 | Rollback | 状态向后流转 |

### 2.4 审计相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 审计日志 | Audit Log | 用户操作的历史记录 |
| 操作时间 | Timestamp | 操作发生的时间 |
| 操作人 | Operator | 执行操作的用户 |
| IP地址 | IP Address | 操作来源的网络地址 |
| 请求方法 | HTTP Method | HTTP请求类型（GET/POST/PUT/DELETE） |
| 状态码 | Status Code | HTTP响应状态码 |

### 2.5 API相关

| 术语 | 英文 | 定义 |
|------|------|------|
| API | Application Programming Interface | 应用程序编程接口 |
| RESTful | Representational State Transfer | 一种API设计风格 |
| 端点 | Endpoint | API的访问地址 |
| 请求体 | Request Body | 请求中携带的数据 |
| 响应体 | Response Body | 响应返回的数据 |
| 错误码 | Error Code | 标识错误类型的代码 |

## 三、标准术语

### 3.1 混凝土检测术语

| 术语 | 符号 | 定义 |
|------|------|------|
| 抗压强度 | fc | 混凝土试件单位面积承受的压力 |
| 抗折强度 | ff | 混凝土试件弯曲破坏时的强度 |
| 坍落度 | SL | 新拌混凝土的流动性指标 |
| 标准养护 | - | 温度20±2°C，相对湿度95%以上的养护条件 |
| 同条件养护 | - | 与实际构件相同环境条件下的养护 |
| 强度等级 | Cxx | 混凝土强度等级，如C30表示标准值30MPa |
| 尺寸修正系数 | η | 非标准尺寸试件的强度修正系数 |

### 3.2 钢筋检测术语

| 术语 | 符号 | 定义 |
|------|------|------|
| 屈服强度 | ReL | 钢筋开始塑性变形时的应力 |
| 抗拉强度 | Rm | 钢筋断裂前承受的最大应力 |
| 断后伸长率 | A | 断裂后标距的伸长百分比 |
| 最大力总伸长率 | Agt | 最大力时的伸长率 |
| 强屈比 | Rm/ReL | 抗拉强度与屈服强度的比值 |
| 弯曲性能 | - | 钢筋在规定角度弯曲后不产生裂纹的能力 |
| 公称直径 | d | 钢筋的标准直径 |
| 横截面积 | S | 钢筋的截面面积 |

### 3.3 沥青检测术语

| 术语 | 符号 | 定义 |
|------|------|------|
| 针入度 | Pen | 标准条件下针入沥青的深度，表示软硬程度 |
| 软化点 | SP | 沥青软化时的温度 |
| 延度 | D | 沥青拉伸断裂时的长度 |
| 马歇尔稳定度 | MS | 沥青混合料的抗压强度 |
| 流值 | FL | 马歇尔试验时的变形量 |
| 沥青含量 | - | 混合料中沥青的质量百分比 |
| 空隙率 | VV | 混合料中空隙的体积百分比 |

### 3.4 土工检测术语

| 术语 | 符号 | 定义 |
|------|------|------|
| 含水率 | w | 土中水的质量与干土质量之比 |
| 密度 | ρ | 单位体积土的质量 |
| 干密度 | ρd | 单位体积土中固体颗粒的质量 |
| 最大干密度 | ρd,max | 击实试验得到的最大干密度 |
| 最优含水率 | wop | 击实试验得到最大干密度时的含水率 |
| 压实度 | K | 实际干密度与最大干密度之比 |
| 承载比 | CBR | 加州承载比，表示土的承载力 |
| 液限 | wL | 土从可塑状态转为流动状态的界限含水率 |
| 塑限 | wP | 土从可塑状态转为半固态的界限含水率 |
