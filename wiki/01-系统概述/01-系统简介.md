# 系统简介

## 一、系统定位

本系统是专为**浦东国际机场四期扩建工程工地试验室**开发的实验室信息管理系统（Laboratory Information Management System，简称LIMS），旨在实现检测业务全流程数字化管理，满足第三方检测服务的规范化、标准化要求。

### 1.1 建设目标

- **业务流程标准化**：实现委托→样品→检测→报告的完整闭环管理
- **数据管理规范化**：确保检测数据的完整性、可追溯性
- **质量控制体系化**：支持ISO/IEC 17025、CNAS、CMA认证要求
- **管理决策智能化**：提供统计分析、预警监控功能

### 1.2 适用范围

本系统适用于机场工程工地试验室的以下检测领域：

| 检测领域 | 主要检测项目 | 执行标准 |
|----------|-------------|----------|
| 混凝土 | 抗压强度、抗折强度、坍落度、凝结时间、含气量 | GB/T 50081、GB/T 50080 |
| 钢筋 | 屈服强度、抗拉强度、伸长率、弯曲性能、反向弯曲 | GB/T 228.1、GB/T 232 |
| 沥青 | 针入度、软化点、延度、马歇尔稳定度、沥青含量 | JTG E20、JTG F40 |
| 土工 | 含水率、密度、压实度、承载比(CBR)、无侧限抗压强度 | JTG 3430、JTG E40 |
| 骨料 | 级配、含泥量、压碎值、针片状含量、密度 | JTG 3432、GB/T 14684 |

| 特性 | 说明 |

## 五、系统部署

### 5.1 部署方式

系统支持以下部署方式：

1. **Docker部署（推荐）**
   ```bash
   docker-compose up -d
   ```

2. **传统部署**
   - 后端：Gunicorn + Nginx
   - 前端：Nginx静态托管
   - 数据库：PostgreSQL主从
   - 缓存：Redis Sentinel
