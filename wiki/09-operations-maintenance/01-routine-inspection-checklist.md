# 例行巡检清单

## 每日快速项

- [ ] 站点首页与登录可用
- [ ] API 抽样 200，无持续 5xx
- [ ] 磁盘使用率 < 85%
- [ ] 数据库连接正常
- [ ] 错误日志无新增持续异常栈

## 每周

- 备份成功记录
- 证书有效期（若 30 天内到期则告警）
