# 人员管理

本文档介绍人员档案、证书、授权、培训和能力评价的管理操作。

---

## 一、人员档案

### 1.1 人员列表

**路径**：人员管理 → 人员档案

### 1.2 人员信息

| 字段 | 说明 |
|------|------|
| 工号 | 人员工号 |
| 姓名 | 人员姓名 |
| 部门 | 所属部门 |
| 岗位 | 岗位职务 |
| 学历 | 最高学历 |
| 专业 | 所学专业 |
| 入职日期 | 入职时间 |
| 状态 | 在职/离职 |

### 1.3 新增人员

1. 点击「新增」按钮
2. 填写人员基本信息
3. 上传照片
4. 保存

## 二、证书管理

### 2.1 证书类型

| 类型 | 说明 |
|------|------|
| 上岗证 | 检测上岗证书 |
| 资格证 | 职业资格证书 |
| 培训证 | 培训合格证书 |
| 其他 | 其他证书 |

### 2.2 添加证书

1. 进入人员详情页
2. 点击「证书」标签页
3. 点击「添加证书」
4. 填写证书信息：

| 字段 | 说明 |
|------|------|
| 证书类型 | 证书类型 |
| 证书名称 | 证书名称 |
| 证书编号 | 证书编号 |
| 发证机构 | 发证单位 |
| 发证日期 | 发证日期 |
| 有效期 | 证书有效期 |
| 证书附件 | 证书扫描件 |

### 2.3 证书到期提醒

系统自动监控证书有效期：
- 到期前30天发送提醒
- 过期证书标记为无效

## 三、人员授权

### 3.1 授权内容

授权检测人员可执行的检测项目：
- 检测类别
- 检测方法
- 检测参数

### 3.2 添加授权

1. 进入人员详情页
2. 点击「授权」标签页
3. 点击「添加授权」
4. 选择授权内容：
   - 检测类别
   - 检测方法
   - 授权日期
   - 有效期
5. 保存

### 3.3 授权审批

人员授权需经技术负责人审批：
1. 提交授权申请
2. 技术负责人审核
3. 审批通过后生效

## 六、常见问题

### Q1: 人员离职后数据如何处理？

**答案**：将人员状态设为「离职」，历史数据保留可查询。

### Q2: 如何查看人员的授权范围？

**方法**：进入人员详情页，查看「授权」标签页。

### Q3: 证书过期后还能进行检测吗？

**答案**：不可以。证书过期后相关授权自动失效。
