# 模板管理

本文档介绍原始记录模板和报告模板的管理操作。

---

## 一、原始记录模板

### 1.1 模板列表

**路径**：质量管理 → 原始记录模板

### 1.2 模板信息

| 字段 | 说明 |
|------|------|
| 模板编号 | 模板唯一标识 |
| 模板名称 | 模板名称 |
| 检测方法 | 关联的检测方法 |
| 版本号 | 模板版本 |
| 状态 | 启用/停用 |

### 1.3 创建模板

1. 点击「新增」按钮
2. 填写模板基本信息
3. 设计模板结构
4. 保存

### 1.4 模板结构设计

模板由以下部分组成：

#### 基本信息区
- 任务编号
- 样品编号
- 检测日期
- 环境条件

#### 设备信息区
- 设备名称
- 设备编号
- 校准有效期

#### 检测数据区
- 输入字段
- 计算字段
- 结果字段

#### 判定区
- 判定规则
- 判定结论

### 1.5 字段配置

| 字段属性 | 说明 |
|----------|------|
| 字段名称 | 显示名称 |
| 字段类型 | 文本/数字/日期/选择 |
| 是否必填 | 必填/选填 |
| 默认值 | 默认值 |
| 单位 | 计量单位 |
| 精度 | 小数位数 |
| 最小值 | 数值下限 |
| 最大值 | 数值上限 |

### 1.6 计算公式

支持配置自动计算公式：

```
示例：抗压强度 = 荷载 × 1000 / 面积
```

## 二、报告模板

### 2.1 模板列表

**路径**：质量管理 → 报告模板

### 2.2 报告模板类型

| 类型 | 说明 |
|------|------|
| 检测报告 | 常规检测报告 |
| 检测记录 | 原始记录格式 |
| 证书 | 合格证书 |

### 2.3 报告模板内容

报告模板包含：
- 封面
- 检测信息
- 检测结果表格
- 判定结论
- 签字栏
- 备注

### 2.4 模板变量

可在模板中使用变量：

| 变量 | 说明 |
|------|------|
| {{report_no}} | 报告编号 |
| {{commission_no}} | 委托单号 |
| {{project_name}} | 项目名称 |
| {{client_name}} | 委托单位 |
| {{test_date}} | 检测日期 |
| {{result}} | 检测结果 |

## 三、模板版本管理

### 3.1 版本控制

模板支持版本管理：
- 新建版本
- 版本对比
- 版本回退

### 3.2 版本发布

1. 编辑模板
2. 测试验证
3. 发布新版本
4. 旧版本自动停用

---

## 四、常见问题

### Q1: 如何修改正在使用的模板？

**方法**：创建新版本，不影响正在使用的记录。

### Q2: 模板可以复制吗？

**方法**：可以。点击「复制」按钮，基于现有模板创建新模板。

### Q3: 如何测试模板效果？

**方法**：使用「预览」功能，查看模板渲染效果。
