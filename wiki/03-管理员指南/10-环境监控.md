# 环境监控

本文档介绍环境监控点位的配置和报警管理。

---

## 一、监控点位管理

### 1.1 点位列表

**路径**：环境监控 → 监控点位

### 1.2 点位信息

| 字段 | 说明 |
|------|------|
| 点位编号 | 点位唯一标识 |
| 点位名称 | 点位名称 |
| 位置描述 | 具体位置 |
| 监控类型 | 温度/湿度/温湿度 |
| 温度下限 | 温度报警下限 |
| 温度上限 | 温度报警上限 |
| 湿度下限 | 湿度报警下限 |
| 湿度上限 | 湿度报警上限 |
| 状态 | 启用/停用 |

### 1.3 新增监控点位

1. 点击「新增」按钮
2. 填写点位信息
3. 设置报警阈值
4. 保存

---

## 二、环境记录

### 2.1 记录查看

**路径**：环境监控 → 环境记录

### 2.2 记录信息

| 字段 | 说明 |
|------|------|
| 记录时间 | 记录时间 |
| 点位名称 | 监控点位 |
| 温度 | 温度值 |
| 湿度 | 湿度值 |
| 是否报警 | 是否超限 |

### 2.3 数据采集

环境数据可通过以下方式采集：
- 自动采集：连接传感器自动采集
- 手动录入：人工测量后录入

---

## 三、报警管理

### 3.1 报警列表

**路径**：环境监控 → 环境报警

### 3.2 报警类型

| 类型 | 说明 |
|------|------|
| 温度偏高 | 温度超过上限 |
| 温度偏低 | 温度低于下限 |
| 湿度偏高 | 湿度超过上限 |
| 湿度偏低 | 湿度低于下限 |

### 3.3 报警处理

1. 查看报警信息
2. 确认报警原因
3. 采取处理措施
4. 记录处理结果
5. 关闭报警

---

## 四、数据统计

### 4.1 温湿度曲线

可查看指定时间段的温湿度变化曲线图。

### 4.2 统计报表

- 日统计报表
- 周统计报表
- 月统计报表

---

## 五、常见问题

### Q1: 如何修改报警阈值？

**方法**：编辑监控点位信息，修改温度/湿度上下限。

### Q2: 报警通知如何配置？

**方法**：在系统设置中配置报警通知接收人。

### Q3: 历史数据保存多久？

**答案**：默认保存1年，可在系统配置中调整。
