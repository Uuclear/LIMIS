# 标准库管理

本文档介绍标准规范、检测方法和参数库的管理操作。

---

## 一、标准管理

### 1.1 标准列表

**路径**：标准规范 → 标准列表

### 1.2 标准信息

| 字段 | 说明 |
|------|------|
| 标准编号 | 标准号，如GB/T 50081 |
| 标准名称 | 标准名称 |
| 标准类型 | 国家/行业/地方/企业/国际 |
| 状态 | 现行/即将实施/已废止 |
| 发布日期 | 发布日期 |
| 实施日期 | 实施日期 |
| 适用领域 | 混凝土/钢筋/沥青/土工等 |

### 1.3 新增标准

1. 点击「新增」按钮
2. 填写标准信息
3. 上传标准文件
4. 保存

### 1.4 标准状态管理

| 状态 | 说明 |
|------|------|
| 现行 | 当前有效 |
| 即将实施 | 即将生效 |
| 已废止 | 已废止失效 |
| 草案 | 草案阶段 |

---

## 二、检测方法管理

### 2.1 检测方法列表

**路径**：标准规范 → 检测方法

### 2.2 检测方法信息

| 字段 | 说明 |
|------|------|
| 方法编号 | 方法编号 |
| 方法名称 | 检测方法名称 |
| 所属标准 | 关联的标准 |
| 检测类别 | 混凝土/钢筋/沥青/土工 |
| 是否有效 | 有效/无效 |

### 2.3 新增检测方法

1. 点击「新增」按钮
2. 填写方法信息
3. 关联检测参数
4. 保存

---

## 三、参数库管理

### 3.1 检测参数列表

**路径**：标准规范 → 参数库

### 3.2 参数信息

| 字段 | 说明 |
|------|------|
| 参数编号 | 参数编号 |
| 参数名称 | 参数名称 |
| 所属方法 | 关联的检测方法 |
| 单位 | 计量单位 |
| 精度 | 小数位数 |
| 判定规则 | 合格判定规则 |

### 3.3 判定规则配置

| 规则类型 | 说明 |
|----------|------|
| 数值判定 | 与标准值比较 |
| 范围判定 | 是否在允许范围内 |
| 等级判定 | 根据强度等级判定 |

---

## 四、方法验证

### 4.1 验证目的

验证实验室是否具备正确执行检测方法的能力。

### 4.2 添加验证记录

1. 进入标准详情页
2. 点击「方法验证」标签页
3. 点击「添加验证」
4. 填写验证信息：

| 字段 | 说明 |
|------|------|
| 验证日期 | 验证执行日期 |
| 验证人 | 执行验证的人员 |
| 验证范围 | 验证的范围内容 |
| 验证方法 | 验证采用的方法 |
| 验证结论 | 通过/不通过 |
| 验证报告 | 验证报告附件 |

---

## 五、常见问题

### Q1: 标准废止后相关检测方法还能用吗？

**答案**：不可以。标准废止后关联的检测方法自动标记为无效。

### Q2: 如何查看标准是否现行有效？

**方法**：在标准列表查看状态字段，或进入标准详情页查看。

### Q3: 如何添加新的检测参数？

**方法**：先添加检测方法，再在方法下添加检测参数。
