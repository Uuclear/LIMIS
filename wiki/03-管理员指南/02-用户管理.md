# 用户管理

本文档介绍用户管理的操作方法，包括用户创建、角色分配、权限配置等。

---

## 一、用户列表

### 1.1 查看用户列表

**路径**：系统管理 → 用户管理

用户列表显示所有系统用户。

### 1.2 筛选条件

| 筛选项 | 说明 |
|--------|------|
| 用户名 | 模糊搜索 |
| 姓名 | 模糊搜索 |
| 部门 | 下拉选择 |
| 状态 | 启用/禁用 |
| 角色 | 下拉选择 |

### 1.3 列表字段

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名 |
| 姓名 | 真实姓名 |
| 部门 | 所属部门 |
| 角色 | 分配的角色 |
| 状态 | 启用/禁用 |
| 最后登录 | 最后登录时间 |

---

## 二、创建用户

### 2.1 创建步骤

1. 点击「新增」按钮
2. 填写用户信息
3. 分配角色
4. 点击「保存」

### 2.2 用户信息字段

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 用户名 | 是 | 登录用户名，创建后不可修改 |
| 姓名 | 是 | 真实姓名 |
| 密码 | 是 | 初始密码，建议用户首次登录后修改 |
| 手机号 | 否 | 联系电话 |
| 邮箱 | 否 | 电子邮箱 |
| 部门 | 否 | 所属部门 |
| 职称 | 否 | 职称职务 |
| 状态 | 是 | 启用/禁用，默认启用 |
| 角色 | 否 | 可分配多个角色 |

### 2.3 密码设置

初始密码要求：
- 至少8个字符
- 包含大小写字母、数字、特殊字符中的至少3种
- 不能与用户名相同

### 2.4 角色分配

可为用户分配一个或多个角色：
- 勾选需要的角色
- 用户权限为所有角色权限的并集
- 建议按最小权限原则分配

---

## 三、编辑用户

### 3.1 编辑步骤

1. 在用户列表找到目标用户
2. 点击「编辑」按钮
3. 修改需要更新的信息
4. 点击「保存」

### 3.2 可修改字段

- 姓名
- 手机号
- 邮箱
- 部门
- 职称
- 状态
- 角色

### 3.3 不可修改字段

- 用户名（创建后不可修改）

---

## 四、重置密码

### 4.1 重置场景

以下情况需要重置密码：
- 用户忘记密码
- 用户账号被锁定
- 安全审计要求

### 4.2 重置步骤

1. 在用户列表找到目标用户
2. 点击「重置密码」按钮
3. 输入新密码
4. 确认重置
5. 通知用户新密码

### 4.3 重置后操作

- 用户需使用新密码登录
- 建议用户登录后立即修改密码
- 系统记录密码重置操作

---

## 五、启用/禁用用户

### 5.1 禁用用户

禁用后用户无法登录系统：

1. 在用户列表找到目标用户
2. 点击「禁用」按钮
3. 确认禁用

### 5.2 启用用户

1. 在用户列表找到目标用户
2. 点击「启用」按钮
3. 确认启用

### 5.3 批量操作

可批量启用或禁用多个用户：
1. 勾选多个用户
2. 点击批量操作按钮
3. 选择启用或禁用
4. 确认操作

---

## 六、角色管理

### 6.1 查看角色列表

**路径**：系统管理 → 角色管理

### 6.2 系统预设角色

| 角色 | 代码 | 说明 |
|------|------|------|
| 系统管理员 | admin | 拥有全部权限 |
| 技术负责人 | tech_director | 技术管理、报告批准 |
| 质量负责人 | quality_director | 质量管理、内审管评 |
| 授权签字人 | auth_signer | 报告签署 |
| 审核人 | reviewer | 报告审核、记录复核 |
| 监督员 | supervisor | 质量监督 |
| 检测人员 | tester | 检测执行 |
| 样品管理员 | sample_clerk | 样品管理 |
| 设备管理员 | equip_manager | 设备管理 |
| 接待员 | reception | 委托登记 |
| 委托方 | client | 委托提交 |

### 6.3 创建自定义角色

1. 点击「新增角色」
2. 填写角色信息：
   - 角色名称
   - 角色编码
   - 描述
3. 分配权限
4. 保存

### 6.4 编辑角色权限

1. 点击角色的「编辑权限」
2. 勾选需要的权限
3. 保存

权限按模块分组：
- 系统管理
- 项目管理
- 委托管理
- 样品管理
- 检测业务
- 报告管理
- 设备管理
- 人员管理
- 质量管理
- 标准规范
- 耗材管理

每个模块可分配的操作权限：
- 查看 (view)
- 创建 (create)
- 编辑 (edit)
- 删除 (delete)
- 审批 (approve)
- 导出 (export)

## 七、权限管理

### 7.1 权限粒度

系统支持三级权限控制：

| 级别 | 说明 |
|------|------|
| 模块级 | 是否可访问某个功能模块 |
| 操作级 | 可执行的操作类型 |
| 数据级 | 可访问的数据范围 |

### 7.2 查看用户权限

1. 进入用户详情页
2. 点击「权限」标签页
3. 显示该用户所有权限

### 7.3 权限继承

- 用户权限 = 所有角色权限的并集
- 超级管理员拥有全部权限
- 权限变更立即生效

## 八、用户导入导出

### 8.1 导出用户

1. 点击「导出」按钮
2. 选择导出格式（Excel）
3. 下载文件

### 8.2 批量导入用户

1. 下载导入模板
2. 填写用户信息
3. 点击「导入」按钮
4. 选择填写好的文件
5. 确认导入

### 8.3 导入注意事项

- 用户名不能重复
- 必填字段不能为空
- 角色需填写角色编码
- 导入失败会显示错误信息

## 九、常见问题

### Q1: 用户名可以修改吗？

**答案**：不可以。用户名创建后不可修改，如需更换请创建新用户。

### Q2: 用户忘记密码怎么办？

**方法**：管理员可在用户管理页面重置密码。

### Q3: 如何限制用户只能查看自己的数据？

**方法**：通过数据级权限配置，设置用户的数据访问范围。

### Q4: 用户被锁定怎么办？

**方法**：管理员可重置密码解锁，或等待锁定时间自动解锁。

### Q5: 如何批量修改用户角色？

**方法**：使用批量编辑功能，选择多个用户后统一修改角色。
