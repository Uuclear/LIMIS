# 质量管理

本文档介绍内部审核、管理评审、不符合项、投诉等质量管理活动的操作。

---

## 一、内部审核

### 1.1 审核计划

**路径**：质量管理 → 内部审核

### 1.2 创建审核计划

1. 点击「新增」按钮
2. 填写审核信息：

| 字段 | 说明 |
|------|------|
| 审核编号 | 审核唯一标识 |
| 审核主题 | 审核主题 |
| 审核类型 | 例行/专项/跟踪 |
| 审核范围 | 审核范围描述 |
| 计划日期 | 计划审核日期 |
| 审核组长 | 审核组长 |
| 审核员 | 审核组成员 |

### 1.3 审核执行

1. 开始审核
2. 记录审核发现
3. 填写审核结论
4. 提交审核报告

### 1.4 审核发现

| 类型 | 说明 |
|------|------|
| 不符合项 | 不满足要求的问题 |
| 观察项 | 需关注的问题 |
| 改进建议 | 改进建议 |

### 1.5 纠正措施

对审核发现的问题制定纠正措施：

| 字段 | 说明 |
|------|------|
| 问题描述 | 问题描述 |
| 原因分析 | 根本原因分析 |
| 纠正措施 | 纠正措施计划 |
| 责任人 | 责任人 |
| 完成期限 | 完成期限 |
| 验证结果 | 措施验证结果 |

## 三、不符合项管理

### 3.1 不符合项来源

| 来源 | 说明 |
|------|------|
| 内部发现 | 内部检查发现 |
| 内部审核 | 内审发现 |
| 客户投诉 | 客户投诉 |
| 能力验证 | 能力验证结果 |

### 3.2 登记不符合项

1. 点击「新增」按钮
2. 填写不符合项信息：

| 字段 | 说明 |
|------|------|
| 发现日期 | 发现日期 |
| 发现来源 | 发现来源 |
| 问题描述 | 问题描述 |
| 涉及条款 | 涉及的标准条款 |
| 严重程度 | 严重/一般 |

### 3.3 处理流程

1. 原因分析
2. 制定纠正措施
3. 实施纠正措施
4. 验证措施有效性
5. 关闭不符合项

## 五、能力验证

### 5.1 能力验证计划

**路径**：质量管理 → 能力验证

### 5.2 登记能力验证

| 字段 | 说明 |
|------|------|
| 验证编号 | 验证编号 |
| 验证项目 | 验证项目 |
| 组织机构 | 组织机构 |
| 验证日期 | 验证日期 |
| 验证结果 | 满意/有问题/不满意 |

### 5.3 结果处理

- **满意**：能力确认
- **有问题**：分析原因，采取纠正措施
- **不满意**：停止相关检测，整改后重新验证

## 七、常见问题

### Q1: 内审发现的问题如何跟踪？

**方法**：在审核详情页查看纠正措施的执行状态。

### Q2: 管理评审多久进行一次？

**答案**：建议每年至少进行一次管理评审。

### Q3: 不符合项多久需要关闭？

**答案**：根据严重程度确定，一般不超过30天。
