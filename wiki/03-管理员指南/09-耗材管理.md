# 耗材管理

本文档介绍耗材的入库、出库和库存管理操作。

---

## 一、供应商管理

### 1.1 供应商列表

**路径**：耗材管理 → 供应商

### 1.2 供应商信息

| 字段 | 说明 |
|------|------|
| 供应商名称 | 供应商名称 |
| 联系人 | 联系人姓名 |
| 联系电话 | 联系电话 |
| 地址 | 供应商地址 |
| 评价得分 | 供应商评价 |
| 是否合格 | 合格供应商标识 |

### 1.3 新增供应商

1. 点击「新增」按钮
2. 填写供应商信息
3. 保存

---

## 二、耗材台账

### 2.1 耗材列表

**路径**：耗材管理 → 耗材列表

### 2.2 耗材信息

| 字段 | 说明 |
|------|------|
| 耗材编号 | 耗材唯一标识 |
| 耗材名称 | 耗材名称 |
| 规格型号 | 规格型号 |
| 单位 | 计量单位 |
| 库存数量 | 当前库存 |
| 安全库存 | 最低库存警戒线 |
| 有效期 | 有效期 |
| 存放位置 | 存放位置 |

### 2.3 新增耗材

1. 点击「新增」按钮
2. 填写耗材信息
3. 设置安全库存
4. 保存

### 2.4 库存预警

当库存低于安全库存时，系统自动发送预警通知。

---

## 三、入库管理

### 3.1 入库登记

**路径**：耗材管理 → 入库记录

### 3.2 入库操作

1. 点击「新增入库」
2. 选择耗材
3. 填写入库信息：

| 字段 | 说明 |
|------|------|
| 入库日期 | 入库日期 |
| 供应商 | 供应商 |
| 批次号 | 批次号 |
| 入库数量 | 入库数量 |
| 单价 | 单价 |
| 生产日期 | 生产日期 |
| 有效期 | 有效期 |
| 验收人 | 验收人 |

4. 保存

### 3.3 入库审核

入库记录需审核后库存才会更新：
1. 提交入库申请
2. 管理员审核
3. 审核通过后库存增加

---

## 四、出库管理

### 4.1 出库登记

**路径**：耗材管理 → 出库记录

### 4.2 出库操作

1. 点击「新增出库」
2. 选择耗材
3. 填写出库信息：

| 字段 | 说明 |
|------|------|
| 出库日期 | 出库日期 |
| 出库数量 | 出库数量 |
| 用途 | 使用用途 |
| 领用人 | 领用人 |
| 备注 | 其他说明 |

4. 保存

### 4.3 出库规则

- 出库数量不能超过库存数量
- 优先出库即将过期的耗材（先进先出）
- 出库后库存自动减少

---

## 五、库存盘点

### 5.1 盘点操作

1. 导出库存清单
2. 实地盘点
3. 记录实际数量
4. 录入盘点结果
5. 处理盘盈盘亏

### 5.2 盘点差异处理

| 差异类型 | 处理方式 |
|----------|----------|
| 盘盈 | 入库调整 |
| 盘亏 | 出库调整 |

---

## 六、常见问题

### Q1: 如何查看耗材的出入库历史？

**方法**：进入耗材详情页，查看「出入库记录」标签页。

### Q2: 耗材过期了怎么办？

**处理**：将过期耗材标记为不可用，进行报废处理。

### Q3: 如何设置安全库存？

**方法**：编辑耗材信息，设置「安全库存」字段。
