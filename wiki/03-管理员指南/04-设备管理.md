# 设备管理

本文档介绍仪器设备的管理操作，包括设备台账、校准管理、期间核查和维护保养。

---

## 一、设备台账

### 1.1 设备列表

**路径**：设备管理 → 设备台账

### 1.2 设备信息字段

| 字段 | 说明 |
|------|------|
| 设备编号 | 设备唯一标识 |
| 设备名称 | 设备名称 |
| 规格型号 | 设备规格型号 |
| 设备分类 | A类/B类/C类 |
| 状态 | 在用/停用/送检中/已报废 |
| 校准有效期 | 校准到期日期 |
| 存放位置 | 设备存放地点 |
| 保管人 | 设备保管责任人 |

### 1.3 设备分类

| 分类 | 说明 | 管理要求 |
|------|------|----------|
| A类 | 强制检定设备 | 必须定期检定 |
| B类 | 非强制检定设备 | 需定期校准 |
| C类 | 辅助设备 | 自行管理 |

## 二、设备登记

### 2.1 新增设备

1. 点击「新增」按钮
2. 填写设备信息
3. 上传设备照片
4. 保存

### 2.2 设备信息

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 设备编号 | 自动 | 系统自动生成 |
| 设备名称 | 是 | 设备名称 |
| 规格型号 | 是 | 设备规格型号 |
| 制造商 | 是 | 生产厂家 |
| 出厂编号 | 否 | 出厂序列号 |
| 购置日期 | 是 | 购买日期 |
| 购置金额 | 否 | 购买金额 |
| 设备分类 | 是 | A/B/C类 |
| 存放位置 | 是 | 存放地点 |
| 保管人 | 是 | 保管责任人 |
| 检测方法 | 否 | 可用于的检测方法 |

---

## 三、校准管理

### 3.1 校准记录

**路径**：设备管理 → 校准记录

### 3.2 添加校准记录

1. 进入设备详情页
2. 点击「校准记录」标签页
3. 点击「添加校准」
4. 填写校准信息：

| 字段 | 说明 |
|------|------|
| 校准日期 | 校准执行日期 |
| 校准机构 | 执行校准的机构 |
| 证书编号 | 校准证书编号 |
| 校准结果 | 合格/不合格 |
| 有效期 | 校准有效期 |
| 证书附件 | 上传校准证书 |

### 3.3 校准预警

系统自动监控校准有效期：
- 到期前30天：发送预警通知
- 到期前7天：发送紧急预警
- 已过期：设备标记为不可用

---

## 四、期间核查

### 4.1 核查目的

在两次校准之间验证设备性能是否保持稳定。

### 4.2 添加核查记录

1. 进入设备详情页
2. 点击「期间核查」标签页
3. 点击「添加核查」
4. 填写核查信息：

| 字段 | 说明 |
|------|------|
| 核查日期 | 核查执行日期 |
| 核查方法 | 核查使用的方法 |
| 核查结果 | 合格/不合格 |
| 核查人 | 执行核查的人员 |
| 核查报告 | 核查报告附件 |

## 五、维护保养

### 5.1 维护类型

| 类型 | 说明 |

## 六、设备使用记录

### 6.2 使用记录查询

- 使用时长统计

## 七、设备状态管理

### 7.1 状态类型

| 状态 | 说明 |
|------|------|
| 在用 | 正常使用中 |
| 停用 | 暂时停用 |
| 送检中 | 送检定/校准中 |
| 已报废 | 已报废处理 |

### 7.2 状态变更

1. 进入设备详情页
2. 点击「变更状态」
3. 选择新状态
4. 填写变更原因
5. 确认变更

---

## 八、常见问题

### Q1: 校准过期的设备能用吗？

**答案**：不可以。系统会阻止使用校准过期的设备进行检测。

### Q2: 如何查看设备使用历史？

**方法**：进入设备详情页，查看「使用记录」标签页。

### Q3: 设备报废后数据还能查看吗？

**答案**：可以。报废设备的历史数据仍可查询。
