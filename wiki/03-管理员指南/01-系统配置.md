# 系统配置

本文档介绍系统管理员可配置的各项参数和设置。

---

## 一、系统参数配置

### 1.1 基本参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 系统名称 | 系统显示名称 | LIMS实验室信息管理系统 |
| 系统Logo | 系统标志图片 | - |
| 登录背景 | 登录页面背景图 | - |
| 会话超时 | 用户无操作自动退出时间（分钟） | 30 |
| 文件上传大小限制 | 单个文件最大大小（MB） | 10 |

### 1.2 业务参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 委托编号前缀 | 委托单编号前缀 | WT |
| 样品编号前缀 | 样品编号前缀 | YP |
| 报告编号前缀 | 报告编号前缀 | RP |
| 盲样编号前缀 | 盲样编号前缀 | BL |
| 设备校准预警天数 | 提前多少天预警 | 30 |
| 样品龄期预警天数 | 提前多少天预警 | 3 |
| 留样默认期限 | 默认留样天数 | 28 |

### 1.3 安全参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 密码最小长度 | 密码最少字符数 | 8 |
| 密码有效期 | 密码有效天数（0表示永不过期） | 90 |
| 登录失败锁定次数 | 连续失败多少次锁定账号 | 5 |
| 账号锁定时间 | 锁定多少分钟后解锁（分钟） | 30 |
| 密码历史记录 | 不能与最近几次密码相同 | 5 |

## 二、环境配置

### 2.1 数据库配置

数据库连接配置位于 `.env` 文件或环境变量：

```bash
# PostgreSQL配置
DB_ENGINE=django.db.backends.postgresql
DB_NAME=limis
DB_USER=limis
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=5432
```

### 2.2 Redis配置

```bash
# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=
REDIS_DB=0
```

### 2.3 文件存储配置

```bash
# 文件存储
MEDIA_ROOT=/data/media
STATIC_ROOT=/data/static
```

### 2.4 邮件配置

```bash
# 邮件服务
EMAIL_HOST=smtp.example.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=noreply@example.com
EMAIL_HOST_PASSWORD=your_password
```

## 四、备份配置

### 4.1 数据库备份

建议配置定时备份：

```bash
# 每天凌晨2点备份
0 2 * * * pg_dump -U limis limis > /backup/limis_$(date +\%Y\%m\%d).sql
```

### 4.2 文件备份

定期备份上传的文件：

```bash
# 每周备份文件
0 3 * * 0 tar -czf /backup/media_$(date +\%Y\%m\%d).tar.gz /data/media
```

### 4.3 备份保留策略

| 备份类型 | 保留时间 |
|----------|----------|
| 每日备份 | 7天 |
| 每周备份 | 4周 |
| 每月备份 | 12个月 |

## 六、定时任务

### 6.1 系统定时任务

| 任务 | 执行时间 | 说明 |
|------|----------|------|
| 设备校准预警检查 | 每天 08:00 | 检查即将到期的设备 |
| 样品龄期预警检查 | 每天 08:00 | 检查即将到期的样品 |
| 留样到期检查 | 每天 08:00 | 检查到期留样 |
| 数据统计汇总 | 每天 00:00 | 汇总统计数据 |
| 临时文件清理 | 每天 03:00 | 清理过期临时文件 |

### 6.2 Celery配置

定时任务使用Celery执行：

```python
# celery配置
CELERY_BROKER_URL = 'redis://localhost:6379/1'
CELERY_RESULT_BACKEND = 'redis://localhost:6379/2'
CELERY_ACCEPT_CONTENT = ['json']
CELERY_TASK_SERIALIZER = 'json'
CELERY_RESULT_SERIALIZER = 'json'
CELERY_TIMEZONE = 'Asia/Shanghai'
```

4. 重启服务
5. 验证数据完整性

## 八、安全配置

### 8.1 HTTPS配置

生产环境必须启用HTTPS：

```nginx
server {
    listen 443 ssl;
    server_name limis.example.com;
    
    ssl_certificate /etc/ssl/certs/limis.crt;
    ssl_certificate_key /etc/ssl/private/limis.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

### 8.2 防火墙配置

只开放必要端口：

| 端口 | 服务 | 访问范围 |
|------|------|----------|
| 80/443 | Nginx | 公网 |
| 5432 | PostgreSQL | 内网 |
| 6379 | Redis | 内网 |

### 8.3 安全加固

- 定期更新系统补丁
- 关闭不必要的服务
- 配置入侵检测
- 定期安全审计

## 九、常见问题

### Q1: 如何修改系统名称？

**方法**：修改系统参数配置中的「系统名称」项。

### Q2: 如何调整会话超时时间？

**方法**：修改「会话超时」参数，单位为分钟。

### Q3: 如何配置邮件通知？

**步骤**：
1. 配置邮件服务器参数
2. 测试邮件发送
3. 启用相关通知

### Q4: 如何查看系统运行状态？

**方法**：查看系统监控页面或服务器监控工具。

### Q5: 如何恢复误删的数据？

**步骤**：
1. 从备份恢复数据库
2. 验证数据完整性
3. 必要时联系技术支持
