# 项目管理

本文档介绍工程项目管理的操作方法。

---

## 一、项目列表

### 1.1 查看项目列表

**路径**：项目管理 → 项目列表

### 1.2 项目信息字段

| 字段 | 说明 |
|------|------|
| 项目编号 | 项目唯一标识 |
| 项目名称 | 工程项目名称 |
| 工程类型 | 房建/市政/交通/水利/机场/其他 |
| 建设单位 | 业主单位 |
| 项目状态 | 筹建/在建/完工/停工 |
| 开工日期 | 项目开工日期 |
| 创建时间 | 登记时间 |

---

## 二、创建项目

### 2.1 创建步骤

1. 点击「新增」按钮
2. 填写项目基本信息
3. 添加参建单位
4. 保存项目

### 2.2 项目基本信息

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 项目编号 | 自动 | 系统自动生成 |
| 项目名称 | 是 | 工程项目全称 |
| 工程类型 | 是 | 选择工程类型 |
| 建设单位 | 是 | 业主单位名称 |
| 项目地址 | 否 | 工程地点 |
| 开工日期 | 是 | 计划开工日期 |
| 竣工日期 | 否 | 计划竣工日期 |
| 项目负责人 | 否 | 项目负责人姓名 |
| 联系电话 | 否 | 联系电话 |
| 项目简介 | 否 | 项目概况描述 |

---

## 三、参建单位管理

### 3.1 单位类型

| 类型 | 说明 |
|------|------|
| 建设单位 | 项目业主 |
| 施工单位 | 承建单位 |
| 监理单位 | 监理单位 |
| 设计单位 | 设计单位 |
| 检测单位 | 检测机构 |

### 3.2 添加参建单位

1. 进入项目详情页
2. 点击「添加参建单位」
3. 填写单位信息：
   - 单位名称
   - 单位类型
   - 负责人
   - 联系电话
   - 资质等级
4. 保存

---

## 四、分部分项工程

### 4.1 工程分解结构

项目可分解为分部工程和分项工程：

```
项目
├── 分部工程1
│   ├── 分项工程1-1
│   └── 分项工程1-2
├── 分部工程2
│   ├── 分项工程2-1
│   └── 分项工程2-2
└── ...
```

### 4.2 添加分部分项

1. 进入项目详情页
2. 点击「分部分项」标签页
3. 点击「添加」
4. 填写信息：
   - 编码
   - 名称
   - 上级工程（可选）
   - 工程量
   - 单位

## 五、合同管理

### 5.1 合同信息

| 字段 | 说明 |
|------|------|
| 合同编号 | 合同编号 |
| 合同名称 | 合同名称 |
| 合同类型 | 检测合同/其他 |
| 签订日期 | 合同签订日期 |

| 检测范围 | 检测内容范围 |

2. 点击「合同」标签页

5. 上传合同扫描件
6. 保存

## 七、项目状态管理

### 7.1 状态类型

| 状态 | 说明 |
|------|------|
| 筹建 | 项目筹备阶段 |
| 在建 | 项目施工中 |
| 完工 | 项目已完成 |
| 停工 | 项目暂停 |

### 7.2 状态变更

1. 进入项目详情页
2. 点击「变更状态」
3. 选择新状态
4. 填写变更原因
5. 确认变更

---

## 八、常见问题

### Q1: 项目编号可以修改吗？

**答案**：不可以。项目编号由系统自动生成，确保唯一性。

### Q2: 如何关联委托单？

**答案**：创建委托单时选择项目即可自动关联。

### Q3: 项目完成后如何处理？

**答案**：将项目状态变更为「完工」，相关数据仍可查询。
