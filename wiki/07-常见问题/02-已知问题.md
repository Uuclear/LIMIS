# 已知问题

本文档列出系统已知的技术限制和问题，以及相应的临时解决方案，帮助用户了解系统边界并规避潜在风险。

---

## 一、浏览器兼容性问题

### 1.1 不支持的浏览器

**问题描述**：
系统不支持Internet Explorer浏览器，使用IE访问会出现功能异常。

**影响范围**：
- IE 11及以下版本
- IE兼容模式

**问题表现**：
- 页面无法正常加载
- JavaScript功能失效
- CSS样式错乱
- 表单无法提交

**临时解决方案**：
- 使用Chrome浏览器（推荐）
- 使用Firefox浏览器
- 使用Edge浏览器（Chromium版）
- 使用Safari浏览器（Mac）

**推荐浏览器版本**：

| 浏览器 | 推荐版本 | 下载地址 |
|--------|----------|----------|
| Chrome | 90+ | https://www.google.com/chrome |
| Firefox | 85+ | https://www.mozilla.org/firefox |
| Edge | 90+ | https://www.microsoft.com/edge |
| Safari | 14+ | Mac自带 |

---

### 1.2 Safari日期选择器问题

**问题描述**：
Safari浏览器中，部分日期选择器组件显示异常。

**影响范围**：
- Safari 14以下版本
- iOS Safari

**问题表现**：
- 日期选择器不显示
- 日期格式解析错误
- 选择日期后无法确认

**临时解决方案**：
1. 手动输入日期（格式：YYYY-MM-DD）
2. 升级Safari到最新版本
3. 使用Chrome或Firefox浏览器

**技术说明**：
Safari对部分HTML5日期输入类型支持不完整，系统已做兼容处理，但旧版本仍可能有问题。

---

### 1.3 移动端浏览器限制

**问题描述**：
移动端浏览器部分功能受限。

**影响范围**：
- iOS Safari
- Android Chrome
- 微信内置浏览器

**问题表现**：

| 问题 | 原因 | 临时解决方案 |
|------|------|--------------|
| 文件上传受限 | iOS安全限制 | 使用电脑端上传 |
| 大文件下载失败 | 移动端存储限制 | 分批下载或使用电脑 |
| 打印功能不可用 | 移动端不支持 | 导出PDF后打印 |
| 弹窗显示异常 | 屏幕尺寸限制 | 滚动查看完整内容 |

**建议**：
- 重要操作建议使用电脑端
- 移动端主要用于查看和简单操作
- 复杂表单填写使用电脑端

---

### 1.4 浏览器缓存问题

**问题描述**：
系统更新后，浏览器缓存可能导致页面显示旧版本。

**影响范围**：
- 所有浏览器
- 系统版本更新后

**问题表现**：
- 新功能不显示
- 页面样式异常
- JavaScript错误
- 功能按钮缺失

**临时解决方案**：
1. **强制刷新页面**
   - Windows: Ctrl + F5
   - Mac: Cmd + Shift + R

2. **清除浏览器缓存**
   - Chrome: Ctrl + Shift + Delete
   - Firefox: Ctrl + Shift + Delete
   - Edge: Ctrl + Shift + Delete

3. **使用无痕模式**
   - Chrome: Ctrl + Shift + N
   - Firefox: Ctrl + Shift + P

4. **清除特定网站缓存**
   - F12打开开发者工具
   - Application → Clear storage

## 二、大数据量性能问题

### 2.1 列表查询性能

**问题描述**：
数据量超过一定规模后，列表查询响应变慢。

**影响范围**：
- 委托列表（超过5000条）
- 样品列表（超过10000条）
- 报告列表（超过3000条）
- 操作日志（超过50000条）

**问题表现**：
- 页面加载超过5秒
- 查询超时
- 浏览器卡顿

**临时解决方案**：

1. **使用筛选条件**
   - 添加日期范围筛选
   - 添加状态筛选
   - 添加部门筛选
   - 缩小查询范围

2. **使用搜索功能**
   - 输入关键词精确搜索
   - 使用编号搜索
   - 避免模糊搜索大数据量

3. **分页查看**
   - 不要设置过大的每页数量
   - 建议每页20-50条
   - 使用分页导航

4. **导出后查看**
   - 导出Excel后在本地查看
   - 使用Excel筛选功能
   - 避免在线浏览大数据

**性能优化建议**：

| 数据量 | 建议操作 |
|--------|----------|
| < 1000 | 正常浏览 |
| 1000-5000 | 使用筛选条件 |
| 5000-10000 | 导出后查看 |
| > 10000 | 联系管理员优化 |

---

### 2.2 批量操作限制

**问题描述**：
批量操作数据量过大时，可能出现超时或失败。

**影响范围**：
- 批量导入（超过1000条）
- 批量导出（超过5000条）
- 批量删除（超过100条）
- 批量状态变更（超过50条）

**问题表现**：
- 操作超时
- 部分数据未处理
- 内存溢出错误

**临时解决方案**：

1. **分批操作**
   - 每次操作不超过100条
   - 分多次完成批量操作
   - 等待前一批完成后再继续

2. **减少数据量**
   - 使用筛选条件缩小范围
   - 只操作必要的数据
   - 避免全量操作

3. **错峰操作**
   - 避免在高峰期批量操作
   - 选择系统负载较低时操作
   - 减少与其他用户冲突

**批量操作建议限制**：

| 操作类型 | 建议上限 | 说明 |
|----------|----------|------|
| 批量导入 | 500条/次 | 分批导入 |
| 批量导出 | 3000条/次 | 分批导出 |
| 批量删除 | 50条/次 | 谨慎操作 |
| 批量状态变更 | 30条/次 | 分批处理 |

---

### 2.3 统计报表性能

**问题描述**：
复杂统计报表生成时间较长。

**影响范围**：
- 年度统计报表
- 多维度交叉报表
- 全量数据统计

**问题表现**：
- 报表生成超过30秒
- 页面加载缓慢
- 可能出现超时

**临时解决方案**：

1. **缩小统计范围**
   - 选择特定时间段
   - 选择特定部门
   - 选择特定类型

2. **使用预置报表**
   - 使用系统预设报表
   - 避免自定义复杂报表
   - 分维度查看统计

3. **错峰查看**
   - 非高峰期查看报表
   - 提前生成报表
   - 导出后离线查看

**报表查看建议**：

| 报表类型 | 建议操作 |
|----------|----------|
| 日报/周报 | 正常查看 |
| 月报 | 缩小范围或错峰 |
| 年报 | 导出后查看 |
| 自定义报表 | 简化维度 |

## 三、文件上传限制

### 3.1 文件大小限制

**问题描述**：
系统对上传文件大小有限制，超过限制无法上传。

**限制说明**：

| 文件类型 | 单文件限制 | 说明 |
|----------|------------|------|
| 普通附件 | 50MB | 文档、图片等 |
| 图片文件 | 10MB | JPG、PNG等 |
| 检测报告 | 100MB | PDF报告文件 |
| 原始数据 | 200MB | 检测原始数据 |
| 批量导入 | 5MB | Excel导入文件 |

**问题表现**：
- 上传进度卡住
- 提示"文件过大"
- 上传失败无响应

**临时解决方案**：

1. **压缩文件**
   - 使用ZIP压缩
   - 压缩图片分辨率
   - 删除不必要的内容

2. **分割文件**
   - 大文件分割为多个小文件
   - 分批上传
   - 合并后使用

3. **转换格式**
   - Word转PDF减小体积
   - 高清图片转普通图片
   - 使用更高效的格式

4. **联系管理员**
   - 特殊情况可申请临时放宽
   - 管理员可调整限制配置
   - 大文件可通过其他方式传输

---

### 3.2 文件格式限制

**问题描述**：
系统只支持特定格式的文件上传。

**支持格式**：

| 类型 | 支持格式 | 不支持格式 |
|------|----------|------------|
| 文档 | PDF、DOC、DOCX、XLS、XLSX | EXE、BAT、CMD |
| 图片 | JPG、JPEG、PNG、GIF、BMP | TIFF、RAW、SVG |
| 压缩 | ZIP、RAR | 7Z、TAR |
| 数据 | CSV、TXT | JSON、XML（需转换） |

**问题表现**：
- 提示"格式不支持"
- 文件上传后无法打开
- 预览功能失效

**临时解决方案**：

1. **转换格式**
   - 使用格式转换工具
   - Word/Excel转PDF
   - 特殊图片转PNG/JPG

2. **压缩打包**
   - 不支持的格式打包为ZIP
   - 上传压缩包
   - 下载后解压使用

3. **联系管理员**
   - 申请添加新格式支持
   - 特殊格式特殊处理

---

### 3.3 文件名限制

**问题描述**：
文件名包含特殊字符可能导致上传失败。

**不支持字符**：
- \ / : * ? " < > |
- 中文空格
- 特殊Unicode字符

**问题表现**：
- 上传失败
- 文件无法下载
- 文件无法预览

**临时解决方案**：

1. **重命名文件**
   - 使用英文或数字命名
   - 使用下划线或横线替代空格
   - 避免特殊字符

2. **规范命名**
   - 使用日期+描述命名
   - 例如：2024-01-15_检测报告.pdf
   - 简洁明了

---

### 3.4 并发上传限制

**问题描述**：
同时上传多个文件可能导致失败。

**限制说明**：
- 同时上传文件数：不超过5个
- 同一用户上传队列：不超过10个

**问题表现**：
- 部分文件上传失败
- 上传进度卡住
- 服务器响应慢

**临时解决方案**：

1. **逐个上传**
   - 一个文件上传完成后上传下一个
   - 避免同时选择多个文件

2. **分批上传**
   - 大量文件分批上传
   - 每批不超过5个
   - 等待前批完成

3. **错峰上传**
   - 避开高峰期
   - 选择网络稳定时上传

---

## 四、并发操作限制

### 4.1 数据编辑锁定

**问题描述**：
同一数据被多人同时编辑时，可能出现冲突。

**影响范围**：
- 委托单编辑
- 样品信息编辑
- 原始记录填写
- 报告编制

**问题表现**：
- 提示"数据已被锁定"
- 保存失败
- 数据覆盖丢失

**临时解决方案**：

1. **等待解锁**
   - 查看锁定人和锁定时间
   - 等待对方完成操作
   - 或联系对方协调

2. **联系管理员**
   - 管理员可强制解锁
   - 紧急情况下解锁

3. **协调操作**
   - 团队内协调编辑时间
   - 避免同时编辑同一数据
   - 使用沟通工具协调

**锁定机制说明**：

| 场景 | 锁定时长 | 说明 |
|------|----------|------|
| 编辑页面打开 | 30分钟 | 自动锁定 |
| 表单保存 | 解锁 | 保存后解锁 |
| 页面关闭 | 解锁 | 关闭后解锁 |
| 超时未操作 | 自动解锁 | 30分钟后 |

---

### 4.2 审批流程并发

**问题描述**：
多人同时审批可能导致流程异常。

**影响范围**：
- 委托评审
- 报告审核
- 报告批准

**问题表现**：
- 审批结果不一致
- 流程状态异常
- 审批记录混乱

**临时解决方案**：

1. **协调审批顺序**
   - 确定审批人顺序
   - 按顺序依次审批
   - 使用沟通工具协调

2. **查看审批状态**
   - 审批前查看当前状态
   - 确认是否已被审批
   - 避免重复审批

3. **联系管理员**
   - 异常情况联系管理员
   - 管理员可重置流程状态

---

### 4.3 编号生成并发

**问题描述**：
高并发创建数据时，编号可能出现重复或不连续。

**影响范围**：
- 委托编号
- 样品编号
- 报告编号

**问题表现**：
- 编号重复提示
- 编号不连续
- 编号格式异常

**临时解决方案**：

1. **避免同时创建**
   - 团队协调创建时间
   - 避免多人同时点击创建
   - 分时段创建数据

2. **检查编号**
   - 创建后检查编号是否正确
   - 发现重复及时报告
   - 管理员可修正编号

3. **手动修改**
   - 编号异常时可手动修改
   - 按规则重新编号
   - 确保唯一性

## 五、移动端适配问题

### 5.1 页面布局问题

**问题描述**：
移动端部分页面布局显示不完整。

**影响范围**：
- 复杂表格页面
- 多列布局页面
- 弹窗对话框

**问题表现**：
- 内容显示不全
- 需要横向滚动
- 按钮点击困难

**临时解决方案**：

1. **使用横屏模式**
   - 手机横屏查看
   - 更大的显示区域
   - 表格显示更完整

2. **使用平板设备**
   - 平板显示效果更好
   - 推荐iPad或Android平板

3. **使用电脑端**
   - 复杂页面使用电脑
   - 获得完整功能体验

**移动端适配说明**：

| 功能 | 移动端支持 | 建议 |
|------|------------|------|
| 数据查看 | 支持 | 正常使用 |
| 简单编辑 | 支持 | 正常使用 |
| 复杂表单 | 部分 | 使用电脑 |
| 批量操作 | 不支持 | 使用电脑 |
| 文件上传 | 有限 | 使用电脑 |
| 打印导出 | 不支持 | 使用电脑 |

2. **分步填写**

## 八、问题反馈渠道

### 8.1 反馈方式

| 渠道 | 适用场景 | 响应时间 |
|------|----------|----------|
| 系统内反馈 | 功能问题、Bug报告 | 1-2工作日 |
| 邮件反馈 | 详细问题描述 | 1工作日 |
| 电话反馈 | 紧急问题 | 即时响应 |

### 8.2 反馈内容要求

提交问题时请包含：

1. **问题描述**
   - 问题发生的具体现象
   - 问题的影响范围

2. **环境信息**
   - 操作系统版本
   - 浏览器版本
   - 用户角色

3. **复现步骤**
   - 详细操作步骤
   - 问题发生的时间点

4. **截图或日志**
   - 问题截图
   - 错误信息
   - 相关数据编号

### 8.3 反馈模板

```
【问题类型】Bug/功能建议/其他

【问题描述】
简要描述遇到的问题

【环境信息】
- 操作系统：Windows 10 / macOS / iOS / Android
- 浏览器：Chrome 120 / Firefox 115 / Safari 14
- 用户角色：检测人员 / 管理员 / 其他

【复现步骤】
1. 进入xxx页面
2. 点击xxx按钮
3. xxx操作
4. 问题出现

【期望结果】
应该发生什么

【实际结果】
实际发生了什么

【附件】
截图或文件
```
