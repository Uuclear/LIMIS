# 常见问题FAQ

本文档汇总系统使用过程中的常见问题及解决方法，帮助用户快速定位和解决问题。

---

## 一、登录问题

### 1.1 忘记密码

**问题描述**：用户忘记登录密码，无法登录系统。

**解决步骤**：

1. **自助重置（如已配置）**
   - 在登录页面点击「忘记密码」链接
   - 输入注册时使用的手机号或邮箱
   - 系统发送验证码
   - 输入验证码后设置新密码

2. **联系管理员重置**
   - 联系系统管理员
   - 提供用户名和身份验证信息
   - 管理员在「用户管理」中重置密码
   - 管理员操作路径：系统管理 → 用户管理 → 选择用户 → 重置密码
   - 重置后系统生成临时密码

3. **首次登录后修改**
   - 使用临时密码登录
   - 系统提示修改密码
   - 设置新密码并确认

**密码要求**：
- 长度至少8个字符
- 包含大小写字母、数字、特殊字符中的至少3种
- 不能与最近5次密码相同

---

### 1.2 账号被锁定

**问题描述**：登录时提示"账号已被锁定"或"账号已停用"。

**可能原因**：
- 连续多次输入错误密码（通常5次）
- 账号被管理员手动停用
- 账号超过有效期未登录
- 账号存在安全风险被系统自动锁定

**解决步骤**：

1. **确认锁定原因**
   - 查看登录页面的错误提示信息
   - 联系系统管理员确认账号状态

2. **等待自动解锁**
   - 如果是密码错误导致的锁定
   - 通常30分钟后自动解锁
   - 或联系管理员立即解锁

3. **管理员解锁**
   - 管理员登录系统
   - 进入「系统管理」→「用户管理」
   - 找到被锁定的用户
   - 点击「解锁」按钮
   - 或编辑用户，将状态改为「启用」

4. **重置密码**
   - 如果忘记密码导致锁定
   - 管理员重置密码后重新登录

---

### 1.3 无法登录系统

**问题描述**：输入正确的用户名和密码，但无法登录。

**排查步骤**：

1. **检查网络连接**
   ```
   - 确认电脑已连接网络
   - 尝试访问其他网站确认网络正常
   - 检查是否需要VPN连接
   ```

2. **检查浏览器**
   - 推荐使用Chrome、Firefox、Edge浏览器
   - 清除浏览器缓存和Cookie
   - 尝试使用无痕/隐私模式
   - 尝试更换浏览器

3. **检查地址**
   - 确认访问的是正确的系统地址
   - 检查是否有http/https协议问题
   - 确认端口号是否正确

4. **检查账号状态**
   - 确认账号是否已激活
   - 确认账号是否在有效期内
   - 确认账号是否被停用

5. **检查验证码**
   - 如果启用了验证码，确保输入正确
   - 注意验证码大小写
   - 点击刷新获取新验证码

6. **联系技术支持**
   - 如果以上步骤都无法解决
   - 联系系统管理员或技术支持
   - 提供错误截图和操作描述

---

### 1.4 验证码问题

**问题描述**：验证码不显示、看不清或输入正确但提示错误。

**解决方法**：

| 问题 | 解决方法 |
|------|----------|
| 验证码不显示 | 刷新页面；检查网络；清除缓存 |
| 看不清验证码 | 点击验证码图片刷新；点击"换一张" |
| 输入正确但错误 | 注意大小写；刷新后重新输入 |
| 验证码过期 | 验证码有效期5分钟，超时需刷新 |

## 二、操作问题

### 2.1 无法编辑数据

**问题描述**：点击编辑按钮无反应，或编辑后无法保存。

**可能原因及解决方法**：

| 原因 | 解决方法 |
|------|----------|
| 无编辑权限 | 联系管理员分配相应权限 |
| 数据状态不允许编辑 | 检查数据状态，部分状态不可编辑 |
| 数据被他人锁定 | 等待他人完成操作或联系管理员解锁 |
| 表单验证未通过 | 检查必填项是否填写完整 |
| 网络问题 | 检查网络连接，刷新页面重试 |

**详细排查步骤**：

1. **检查权限**
   - 确认是否有该模块的编辑权限
   - 查看按钮是否为灰色（灰色表示无权限）
   - 联系管理员确认角色权限配置

2. **检查数据状态**
   - 查看数据当前状态标签
   - 以下状态通常不可编辑：
     - 已审批/已批准
     - 已归档
     - 已作废
     - 已完成

3. **检查锁定状态**
   - 数据可能被其他用户正在编辑
   - 系统会显示锁定提示
   - 等待其他用户完成操作
   - 或联系管理员强制解锁

4. **检查表单验证**
   - 查看表单是否有红色提示
   - 确保所有必填项已填写
   - 检查数据格式是否正确
   - 检查数据范围是否有效

---

### 2.2 按钮灰色不可点击

**问题描述**：某些按钮显示为灰色，无法点击操作。

**可能原因**：

1. **权限不足**
   - 当前用户没有该操作的权限
   - 解决：联系管理员分配相应权限

2. **状态限制**
   - 数据当前状态不允许该操作
   - 例如：草稿状态才能删除，已审批状态不能编辑
   - 解决：确认数据状态，按流程操作

3. **前置条件未满足**
   - 操作需要满足特定条件
   - 例如：报告批准前需要先审核通过
   - 解决：完成前置操作

4. **数据限制**
   - 数据存在关联限制
   - 例如：有子项的数据不能直接删除
   - 解决：先处理关联数据

**常见按钮状态说明**：

| 按钮 | 灰色原因 | 解决方法 |
|------|----------|----------|
| 编辑 | 数据已审批/无权限 | 退回后编辑/申请权限 |
| 删除 | 数据已使用/无权限 | 无法删除/申请权限 |
| 提交 | 必填项未填完整 | 完善表单信息 |
| 审批 | 非当前审批人 | 等待或联系管理员 |
| 发放 | 报告未批准 | 先完成批准流程 |

---

### 2.3 表单保存失败

**问题描述**：填写表单后点击保存，提示保存失败或无反应。

**排查步骤**：

1. **检查必填项**
   - 查看表单中带红色星号(*)的字段
   - 确保所有必填项已填写
   - 检查是否有隐藏的必填项

2. **检查数据格式**
   - 日期格式：YYYY-MM-DD
   - 时间格式：HH:mm:ss
   - 数字格式：检查是否输入了非法字符
   - 电话格式：检查是否为有效号码

3. **检查数据范围**
   - 数值是否在允许范围内
   - 日期是否在有效范围内
   - 字符长度是否超限

4. **检查网络连接**
   - 确保网络稳定
   - 检查是否有网络中断
   - 尝试刷新页面重新填写

5. **检查浏览器控制台**
   - 按F12打开开发者工具
   - 查看Console标签是否有错误
   - 查看Network标签请求是否成功

6. **清除缓存重试**
   - 清除浏览器缓存
   - 使用无痕模式重新打开
   - 重新填写表单

---

### 2.4 文件上传失败

**问题描述**：上传文件时提示失败或无反应。

**常见原因及解决方法**：

| 原因 | 解决方法 |
|------|----------|
| 文件过大 | 压缩文件或分批上传，单文件限制50MB |
| 格式不支持 | 检查允许的文件格式，常见：PDF、JPG、PNG、DOC、XLS |
| 网络超时 | 检查网络连接，大文件建议在稳定网络上传 |
| 文件名特殊字符 | 重命名文件，避免特殊字符 |
| 存储空间不足 | 联系管理员清理或扩容 |

**详细解决步骤**：

1. **检查文件大小**
   - 普通附件：不超过50MB
   - 图片文件：不超过10MB
   - 大文件建议压缩后上传

2. **检查文件格式**
   - 文档：PDF、DOC、DOCX、XLS、XLSX
   - 图片：JPG、JPEG、PNG、GIF
   - 其他：ZIP、RAR
   - 不支持：EXE、BAT等可执行文件

3. **检查文件名**
   - 避免使用特殊字符：\ / : * ? " < > |
   - 避免使用中文空格
   - 文件名不宜过长

4. **检查网络状态**
   - 确保网络稳定
   - 大文件建议使用有线网络
   - 避免在网络高峰期上传

---

### 2.5 页面加载缓慢

**问题描述**：页面打开速度慢，数据加载时间长。

**可能原因及优化建议**：

| 原因 | 解决方法 |
|------|----------|
| 网络带宽不足 | 升级网络带宽；使用有线网络 |
| 数据量过大 | 使用筛选条件缩小范围；分页查询 |
| 浏览器缓存过多 | 清除浏览器缓存 |
| 电脑性能不足 | 关闭不必要的程序；升级硬件 |
| 服务器负载高 | 避开高峰期使用；联系管理员优化 |

**用户端优化建议**：

1. **浏览器优化**
   - 使用最新版Chrome或Edge浏览器
   - 定期清除浏览器缓存
   - 关闭不必要的浏览器标签页
   - 禁用不必要的浏览器插件

2. **网络优化**
   - 使用有线网络代替WiFi
   - 关闭占用带宽的应用
   - 检查网络延迟

3. **操作优化**
   - 使用筛选条件缩小数据范围
   - 避免一次性加载大量数据
   - 分批处理数据

## 三、数据问题

### 3.1 找不到数据

**问题描述**：知道数据存在，但在列表中找不到。

**排查步骤**：

1. **检查筛选条件**
   - 查看是否设置了筛选条件
   - 清除所有筛选条件后重新搜索
   - 检查日期范围是否正确

2. **检查搜索条件**
   - 确认搜索关键词是否正确
   - 尝试使用不同的搜索字段
   - 检查是否有多余空格

3. **检查分页**
   - 查看是否在其他页
   - 调整每页显示数量
   - 使用排序功能

4. **检查权限**
   - 确认是否有查看该数据的权限
   - 部分用户只能查看自己创建的数据
   - 联系管理员确认权限范围

5. **检查数据状态**
   - 数据可能被删除或归档
   - 查看是否包含已删除数据
   - 联系管理员确认数据状态

**常见场景**：

| 场景 | 解决方法 |
|------|----------|
| 新创建的数据看不到 | 刷新页面；检查筛选条件 |
| 只能看到自己的数据 | 正常现象，数据权限限制 |
| 历史数据找不到 | 检查日期范围；查看归档数据 |
| 搜索无结果 | 检查关键词拼写；使用模糊搜索 |

---

### 3.2 数据显示错误

**问题描述**：数据显示与预期不符，或显示异常。

**常见问题及解决方法**：

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 数据显示为空 | 数据未加载完成 | 刷新页面等待加载 |
| 数据显示乱码 | 编码问题 | 联系技术支持 |
| 数字显示错误 | 计算逻辑问题 | 检查计算公式；联系管理员 |
| 日期显示错误 | 时区设置问题 | 检查时区设置 |
| 状态显示异常 | 数据同步延迟 | 刷新页面；稍后查看 |

**排查步骤**：

1. **刷新页面**
   - 按F5或点击刷新按钮
   - 清除缓存后刷新（Ctrl+F5）

2. **检查数据源**
   - 确认数据是否正确录入
   - 检查关联数据是否正确

3. **检查计算逻辑**
   - 查看数据是否经过计算
   - 确认计算公式是否正确

4. **联系管理员**
   - 如果确认是系统问题
   - 提供问题数据截图
   - 说明预期结果和实际结果

---

### 3.3 数据重复

**问题描述**：发现系统中存在重复的数据记录。

**可能原因**：

1. **重复提交**
   - 网络延迟导致重复点击提交按钮
   - 解决：等待响应，避免重复点击

2. **数据导入重复**
   - 导入数据时未做重复检查
   - 解决：导入前检查数据，导入后去重

3. **同步重复**
   - 系统同步数据时产生重复
   - 解决：联系管理员处理

**处理方法**：

1. **确认重复数据**
   - 比对数据内容是否完全一致
   - 检查创建时间和创建人

2. **删除重复数据**
   - 保留正确的数据
   - 删除多余的数据
   - 注意：只能删除未使用的数据

3. **联系管理员**
   - 如果数据已关联使用
   - 需要管理员评估后处理

---

### 3.4 数据丢失

**问题描述**：之前保存的数据找不到了。

**排查步骤**：

1. **确认数据是否真的丢失**
   - 检查筛选条件
   - 检查搜索条件
   - 检查数据状态

2. **检查操作日志**
   - 查看是否有删除记录
   - 确认删除操作人和时间

3. **检查数据归档**
   - 数据可能被归档处理
   - 查看归档数据列表

4. **联系管理员**
   - 管理员可查看数据库记录
   - 确认数据是否可恢复

**预防措施**：

- 重要数据及时备份
- 删除操作前确认
- 定期检查数据完整性

---

## 四、打印问题

### 4.1 打印失败

**问题描述**：点击打印按钮后无法打印或打印失败。

**排查步骤**：

1. **检查打印机连接**
   - 确认打印机已开机
   - 检查打印机与电脑连接
   - 检查打印机是否设为默认打印机
   - 检查打印机是否有纸、有墨

2. **检查打印预览**
   - 先查看打印预览
   - 确认内容是否正确显示
   - 检查页面设置

3. **检查浏览器设置**
   - 允许浏览器弹出窗口
   - 允许打印权限
   - 检查是否被拦截

4. **尝试其他方式**
   - 使用浏览器打印功能（Ctrl+P）
   - 导出PDF后打印
   - 更换浏览器尝试

**常见错误及解决方法**：

| 错误提示 | 解决方法 |
|----------|----------|
| 未检测到打印机 | 检查打印机连接和驱动 |
| 打印服务不可用 | 重启打印服务或电脑 |
| 弹出窗口被阻止 | 允许弹出窗口 |
| 打印内容为空 | 检查打印预览；刷新页面 |

---

### 4.2 打印格式问题

**问题描述**：打印出来的内容格式不正确。

**常见问题及解决方法**：

| 问题 | 解决方法 |
|------|----------|
| 内容被截断 | 调整页面边距；选择横向打印 |
| 字体太小/太大 | 调整打印缩放比例 |
| 表格线不显示 | 检查打印设置中的背景图形选项 |
| 页眉页脚不显示 | 检查页面设置 |
| 内容错位 | 调整页面边距和方向 |

**打印设置建议**：

1. **页面设置**
   - 纸张大小：A4
   - 页边距：默认或自定义
   - 页面方向：根据内容选择纵向/横向

2. **打印选项**
   - 背景图形：勾选（显示表格线）
   - 页眉页脚：根据需要选择
   - 缩放：建议100%或适应页面

3. **浏览器打印设置**
   - Chrome：更多设置 → 选项
   - Firefox：页面设置
   - Edge：打印 → 更多设置

---

### 4.3 打印内容不完整

**问题描述**：打印时部分内容缺失或显示不全。

**解决方法**：

1. **检查打印范围**
   - 确认打印内容是否完整
   - 检查是否有隐藏内容

2. **调整页面设置**
   - 增大页边距
   - 调整缩放比例
   - 选择合适的纸张大小

3. **分页打印**
   - 内容过多时分页打印
   - 检查分页是否合理

4. **导出后打印**
   - 导出为PDF格式
   - 使用PDF阅读器打印
   - 可更好地控制打印效果

---

## 五、导出问题

### 5.1 导出失败

**问题描述**：点击导出按钮后无反应或提示失败。

**排查步骤**：

1. **检查数据量**
   - 数据量过大可能导致超时
   - 建议分批导出
   - 使用筛选条件缩小范围

2. **检查网络连接**
   - 确保网络稳定
   - 大数据量导出需要较长时间

3. **检查浏览器设置**
   - 允许下载文件
   - 检查下载路径是否有权限
   - 检查磁盘空间是否充足

4. **检查导出格式**
   - 尝试不同的导出格式
   - Excel格式适合大数据量
   - PDF格式适合打印

**常见错误及解决方法**：

| 错误提示 | 解决方法 |
|----------|----------|
| 导出超时 | 减少数据量；分批导出 |
| 下载失败 | 检查网络；检查下载权限 |
| 文件损坏 | 重新导出；更换格式 |
| 内存不足 | 关闭其他程序；减少数据量 |

---

### 5.2 导出格式问题

**问题描述**：导出的文件格式不正确或内容有问题。

**常见问题及解决方法**：

| 问题 | 解决方法 |
|------|----------|
| 中文乱码 | 使用UTF-8编码打开；用WPS打开 |
| 数字显示为科学计数法 | 设置单元格格式为文本 |
| 日期格式错误 | 在Excel中设置日期格式 |
| 表格线缺失 | 导出时选择带格式选项 |
| 图片未导出 | 检查导出选项；单独导出图片 |

**Excel打开建议**：

1. **乱码问题**
   - 使用Excel导入功能
   - 选择UTF-8编码
   - 或使用WPS打开

2. **格式问题**
   - 导出后调整列宽
   - 设置单元格格式
   - 使用文本格式避免数字转换

3. **大数据量**
   - 使用Excel 2007及以上版本
   - 支持超过65536行数据
   - 或导出为CSV格式

---

### 5.3 导出数据不完整

**问题描述**：导出的数据比页面显示的少，或部分字段缺失。

**可能原因**：

1. **导出范围设置**
   - 检查是否选择了"导出全部"
   - 默认可能只导出当前页

2. **字段权限限制**
   - 部分字段可能无权限导出
   - 联系管理员确认权限

3. **数据筛选**
   - 导出时应用了筛选条件
   - 检查筛选条件是否正确

4. **系统限制**
   - 导出数量有上限
   - 分批导出解决

**解决方法**：

- 确认导出选项设置
- 检查筛选条件
- 分批导出大数据量
- 联系管理员调整导出权限

## 六、权限问题

### 6.1 看不到菜单

**问题描述**：登录后看不到某些功能菜单。

**可能原因及解决方法**：

| 原因 | 解决方法 |
|------|----------|
| 没有对应权限 | 联系管理员分配角色 |
| 角色未激活 | 联系管理员确认角色状态 |
| 菜单被隐藏 | 检查菜单折叠状态 |
| 浏览器问题 | 刷新页面；清除缓存 |

**详细排查步骤**：

1. **确认角色权限**
   - 查看个人中心的角色信息
   - 确认是否分配了相应角色
   - 不同角色有不同的菜单权限

2. **检查菜单状态**
   - 检查左侧菜单是否折叠
   - 点击展开按钮查看完整菜单
   - 检查是否有子菜单

3. **刷新页面**
   - 权限变更后需要刷新
   - 清除缓存后重新登录
   - 检查是否生效

4. **联系管理员**
   - 确认账号角色配置
   - 申请需要的权限
   - 等待权限分配

---

### 6.2 无法操作某功能

**问题描述**：能看到菜单但无法进行某些操作。

**可能原因**：

1. **操作权限不足**
   - 有查看权限但没有操作权限
   - 例如：能查看报告但不能批准

2. **数据权限限制**
   - 只能操作自己创建的数据
   - 不能操作他人的数据

3. **状态限制**
   - 数据当前状态不允许该操作
   - 例如：已审批的数据不能编辑

4. **审批流程限制**
   - 不是当前审批人
   - 需要等待前置审批完成

**解决方法**：

| 场景 | 解决方法 |
|------|----------|
| 无操作权限 | 申请相应权限 |
| 数据权限限制 | 联系数据创建人或管理员 |
| 状态限制 | 按流程操作 |
| 审批流程限制 | 等待或联系当前审批人 |

---

### 6.3 权限申请流程

**问题描述**：需要额外的权限来完成工作。

**申请步骤**：

1. **确认需要的权限**
   - 明确需要访问的模块
   - 明确需要执行的操作
   - 说明申请原因

2. **提交申请**
   - 向直接主管提出申请
   - 或通过系统权限申请功能提交
   - 填写权限申请表

3. **审批流程**
   - 主管审批
   - 系统管理员配置
   - 权限生效

4. **确认权限**
   - 重新登录系统
   - 检查菜单和功能
   - 测试操作权限

**权限申请模板**：

```
申请人：[姓名]
部门：[部门名称]
申请权限：[模块名称] - [操作类型]
申请原因：[详细说明工作需要]
申请期限：[长期/临时]
主管审批：[主管签字]
```

### 6.4 角色权限说明

**常见角色权限对照**：

| 角色 | 主要权限 | 典型操作 |
|------|----------|----------|
| 系统管理员 | 全部权限 | 用户管理、系统配置 |
| 技术负责人 | 技术管理 | 报告批准、任务分配 |
| 质量负责人 | 质量管理 | 内审、管评、不符合项 |
| 授权签字人 | 报告签署 | 报告批准、签名 |
| 审核人 | 审核复核 | 报告审核、记录复核 |
| 检测人员 | 检测执行 | 任务执行、记录填写 |
| 样品管理员 | 样品管理 | 样品登记、流转、处置 |
| 接待员 | 委托接待 | 委托登记、报告发放 |

**权限不足时的提示**：

- 按钮灰色不可点击
- 操作时提示"无权限"
- 菜单不显示
- 数据列表为空

## 七、其他常见问题

### 7.1 页面显示异常

**问题描述**：页面布局错乱、样式丢失或显示异常。

**解决方法**：

1. **清除缓存**
   - Chrome: Ctrl+Shift+Delete
   - 选择清除缓存和Cookie
   - 重新加载页面

2. **强制刷新**
   - Windows: Ctrl+F5
   - Mac: Cmd+Shift+R

3. **检查浏览器**
   - 使用推荐的浏览器版本
   - 禁用浏览器插件
   - 尝试无痕模式

4. **检查网络**
   - 确保CSS/JS文件加载完成
   - 检查网络连接

---

### 7.2 操作超时

**问题描述**：操作过程中提示超时或需要重新登录。

**原因及解决方法**：

| 原因 | 解决方法 |
|------|----------|
| 登录会话过期 | 重新登录系统 |
| 操作时间过长 | 分步保存数据 |
| 网络中断 | 检查网络连接 |
| 服务器超时 | 联系管理员 |

**预防措施**：

- 定期保存表单数据
- 避免长时间停留在编辑页面
- 复杂操作分步完成
- 保持网络连接稳定

---

### 7.3 通知收不到

**问题描述**：系统通知没有收到或延迟收到。

**排查步骤**：

1. **检查通知设置**
   - 进入个人中心
   - 查看通知设置
   - 确认已开启相关通知

2. **检查浏览器通知**
   - 允许浏览器通知权限
   - 检查浏览器通知设置

3. **检查消息中心**
   - 查看系统内消息中心
   - 确认消息是否在列表中

4. **检查邮箱/手机**
   - 检查垃圾邮件
   - 确认联系方式正确

---

### 7.4 附件无法预览

**问题描述**：点击附件无法预览或下载。

**解决方法**：

| 问题 | 解决方法 |
|------|----------|
| 格式不支持 | 下载后使用本地软件打开 |
| 文件损坏 | 联系上传者重新上传 |
| 网络问题 | 检查网络后重试 |
| 权限限制 | 申请文件访问权限 |
