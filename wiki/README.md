## 🏢 系统简介

本系统是专为**浦东国际机场四期扩建工程工地试验室**开发的实验室信息管理系统（LIMS），符合以下认证要求：

- **ISO/IEC 17025** 检测和校准实验室能力认可准则
- **CNAS** 中国合格评定国家认可委员会认可
- **CMA** 检验检测机构资质认定
- **交通工程检测资质** 要求

### 核心业务流程

```
委托登记 → 样品接收 → 检测任务分配 → 原始记录填写 → 报告编制 → 审批发放
```

### 检测领域

| 领域 | 主要检测项目 |
|------|-------------|
| 混凝土 | 抗压强度、抗折强度、坍落度、凝结时间 |
| 钢筋 | 屈服强度、抗拉强度、伸长率、弯曲性能 |
| 沥青 | 针入度、软化点、延度、马歇尔稳定度 |
| 土工 | 含水率、密度、压实度、承载比(CBR) |
| 骨料 | 级配、含泥量、压碎值、针片状含量 |
