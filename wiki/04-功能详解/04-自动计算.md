# 自动计算

本文档详细介绍自动计算公式库的功能和使用方法。

---

## 一、功能概述

系统内置自动计算公式库，支持常见检测参数的自动计算，减少人工计算错误。

### 1.1 支持的检测类型

| 类型 | 计算参数 |
|------|----------|
| 混凝土 | 抗压强度、抗折强度 |
| 钢筋 | 屈服强度、抗拉强度、伸长率、强屈比 |
| 沥青 | 密度、针入度指数 |
| 土工 | 含水率、干密度、压实度 |

---

## 二、混凝土公式

### 2.1 抗压强度计算

**公式**：
```
fc = F × 1000 / A
```

| 参数 | 说明 | 单位 |
|------|------|------|
| fc | 抗压强度 | MPa |
| F | 破坏荷载 | kN |
| A | 受压面积 | mm² |

**尺寸修正**：
- 100mm立方体：修正系数 0.95
- 150mm立方体：修正系数 1.00
- 200mm立方体：修正系数 1.05

### 2.2 抗折强度计算

**公式**：
```
ff = F × L / (b × h²)
```

| 参数 | 说明 | 单位 |
|------|------|------|
| ff | 抗折强度 | MPa |
| F | 破坏荷载 | N |
| L | 支座间距 | mm |
| b | 试件宽度 | mm |
| h | 试件高度 | mm |

---

## 三、钢筋公式

### 3.1 横截面积计算

**公式**：
```
S = π × d² / 4
```

| 参数 | 说明 | 单位 |
|------|------|------|
| S | 横截面积 | mm² |
| d | 公称直径 | mm |

### 3.2 屈服强度计算

**公式**：
```
ReL = Fy × 1000 / S
```

| 参数 | 说明 | 单位 |
|------|------|------|
| ReL | 屈服强度 | MPa |
| Fy | 屈服荷载 | kN |
| S | 横截面积 | mm² |

### 3.3 抗拉强度计算

**公式**：
```
Rm = Fm × 1000 / S
```

| 参数 | 说明 | 单位 |
|------|------|------|
| Rm | 抗拉强度 | MPa |
| Fm | 最大荷载 | kN |
| S | 横截面积 | mm² |

### 3.4 断后伸长率计算

**公式**：
```
A = (Lu - L0) / L0 × 100
```

| 参数 | 说明 | 单位 |
|------|------|------|
| A | 断后伸长率 | % |
| L0 | 原始标距 | mm |
| Lu | 断后标距 | mm |

### 3.5 强屈比计算

**公式**：
```
R = Rm / ReL
```

| 参数 | 说明 |
|------|------|
| R | 强屈比 |
| Rm | 抗拉强度 |
| ReL | 屈服强度 |

**判定要求**：强屈比 ≥ 1.25（抗震钢筋）

## 四、沥青公式

### 4.1 密度计算

**公式**：
```
ρ = m / V
```

| 参数 | 说明 | 单位 |
|------|------|------|
| ρ | 密度 | g/cm³ |
| m | 质量 | g |
| V | 体积 | cm³ |

### 4.2 针入度指数计算

**公式**：
```
PI = (20 - 500 × log Pen × log(SP - 25)) / (50 × log Pen - SP + 25)
```

| 参数 | 说明 | 单位 |
|------|------|------|
| PI | 针入度指数 | - |
| Pen | 25°C针入度 | 0.1mm |
| SP | 软化点 | °C |

---

## 五、土工公式

### 5.1 含水率计算

**公式**：
```
w = (m - md) / md × 100
```

| 参数 | 说明 | 单位 |
|------|------|------|
| w | 含水率 | % |
| m | 湿土质量 | g |
| md | 干土质量 | g |

### 5.2 干密度计算

**公式**：
```
ρd = ρ / (1 + w/100)
```

| 参数 | 说明 | 单位 |
|------|------|------|
| ρd | 干密度 | g/cm³ |
| ρ | 湿密度 | g/cm³ |
| w | 含水率 | % |

### 5.3 压实度计算

**公式**：
```
K = ρd / ρd,max × 100
```

| 参数 | 说明 | 单位 |
|------|------|------|
| K | 压实度 | % |
| ρd | 实际干密度 | g/cm³ |
| ρd,max | 最大干密度 | g/cm³ |

## 六、数值修约

### 6.1 修约规则

系统按照GB/T 8170进行数值修约：

| 精度要求 | 修约规则 |
|----------|----------|
| 0.1 | 保留1位小数 |
| 0.01 | 保留2位小数 |
| 1 | 取整数 |

### 6.2 修约方法

```python
def round_value(value: Decimal, precision: int) -> Decimal:
    """数值修约"""
    return value.quantize(Decimal(f'0.{precision}'), rounding=ROUND_HALF_UP)
```

## 七、公式库实现

### 7.1 公式注册

```python
class FormulaRegistry:
    formulas = {}
    
    def register(self, name: str, formula_class):
        self.formulas[name] = formula_class
    
    def get(self, name: str):
        return self.formulas.get(name)

# 注册公式
registry = FormulaRegistry()
registry.register('concrete_compressive', ConcreteCompressiveFormula)
registry.register('steel_yield', SteelYieldFormula)
```

### 7.2 公式执行

```python
def calculate(formula_name: str, inputs: dict) -> dict:
    """执行计算"""
    formula = registry.get(formula_name)
    if not formula:
        raise ValueError(f'公式 {formula_name} 不存在')
    
    return formula.calculate(inputs)
```

## 八、常见问题

### Q1: 计算结果不正确怎么办？

**排查步骤**：
1. 检查输入数据是否正确
2. 检查单位是否正确
3. 检查公式参数设置
4. 联系技术支持

### Q2: 如何添加新的计算公式？

**方法**：联系系统管理员或开发人员添加新公式。

### Q3: 计算结果可以手动修改吗？

**答案**：可以。但修改后系统会记录修改痕迹。
