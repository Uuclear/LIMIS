# 数据导出

本文档详细介绍系统的数据导出功能。

---

## 一、导出功能概述

系统支持多种数据导出格式，方便数据备份、分析和共享。

### 1.1 支持格式

| 格式 | 扩展名 | 适用场景 |
|------|--------|----------|
| Excel | .xlsx | 数据表格、统计分析 |
| PDF | .pdf | 报告存档、打印 |
| Word | .docx | 报告编辑、修改 |
| CSV | .csv | 数据交换、导入 |

### 1.2 导出范围

- 列表数据导出
- 单条数据导出
- 批量数据导出
- 报告导出

---

## 二、Excel导出

### 2.1 列表导出

**操作步骤**：
1. 进入数据列表页面
2. 设置筛选条件（可选）
3. 点击「导出」按钮
4. 选择导出格式（Excel）
5. 选择导出字段（可选）
6. 确认导出

### 2.2 导出字段选择

可自定义选择导出字段：
- 全选所有字段
- 自定义选择字段
- 使用默认字段配置

### 2.3 导出设置

| 设置项 | 说明 |
|--------|------|
| 文件名 | 自定义导出文件名 |
| 包含标题 | 是否包含列标题 |
| 日期格式 | 日期字段格式 |
| 数字格式 | 数字字段格式 |

### 2.4 大数据量导出

数据量超过10000条时：
- 使用异步导出
- 导出完成后通知下载
- 支持断点续传

---

## 三、报告PDF导出

### 3.1 单份报告导出

**操作步骤**：
1. 进入报告详情页
2. 点击「导出PDF」按钮
3. 选择导出选项
4. 下载PDF文件

### 3.2 批量报告导出

**操作步骤**：
1. 在报告列表勾选多个报告
2. 点击「批量导出」
3. 选择导出格式（PDF）
4. 系统打包下载

### 3.3 PDF选项

| 选项 | 说明 |
|------|------|
| 包含封面 | 是否包含报告封面 |
| 包含签字 | 是否包含电子签名 |
| 包含水印 | 是否添加水印 |
| 页面大小 | A4/A3 |
| 页边距 | 上下左右边距设置 |

---

## 四、报告Word导出

### 4.1 Word导出用途

- 报告模板编辑
- 内容修改调整
- 格式自定义

### 4.2 导出步骤

1. 进入报告详情页
2. 点击「导出Word」按钮
3. 下载Word文件
4. 使用Word编辑

### 4.3 Word模板

系统使用Word模板生成报告：
- 模板位置：`/templates/reports/`
- 模板变量：使用 `{{变量名}}` 格式
- 支持表格、图片、图表

---

## 五、CSV导出

### 5.1 CSV用途

- 数据交换
- 系统间导入导出
- 数据分析

### 5.2 CSV格式

- 编码：UTF-8
- 分隔符：逗号
- 换行：LF
- 引号：文本字段使用双引号包裹

### 5.3 导出步骤

1. 进入数据列表
2. 点击「导出」
3. 选择CSV格式
4. 下载文件

### 6.1 批量选择

## 八、常见问题

### Q1: 导出失败怎么办？

**排查步骤**：
1. 检查网络连接
2. 检查数据量是否过大
3. 尝试减少导出字段
4. 联系管理员

### Q2: 导出文件打不开怎么办？

**解决方法**：
1. 确认文件完整下载
2. 使用正确的软件打开
3. 检查文件编码（CSV）

### Q3: 如何导出所有数据？

**方法**：在列表页选择「全选所有」，然后导出。

### Q4: 导出数据有乱码怎么办？

**解决方法**：
- Excel：使用UTF-8编码打开
- CSV：使用记事本另存为UTF-8
