# 龄期管理

本文档详细介绍龄期管理功能的实现和使用方法。

---

## 一、功能概述

龄期管理用于跟踪混凝土等材料从成型到检测的时间间隔，确保在规定龄期进行检测。

### 1.1 常见龄期

| 龄期 | 说明 |
|------|------|
| 3天 | 早期强度检测 |
| 7天 | 中期强度检测 |
| 28天 | 标准龄期强度 |
| 60天 | 长期强度检测 |
| 90天 | 长期强度检测 |

### 1.2 龄期计算

龄期 = 检测日期 - 成型日期

---

## 二、龄期设置

### 2.1 样品登记时设置

在样品登记时可设置龄期要求：

| 字段 | 说明 |
|------|------|
| 成型日期 | 样品成型日期 |
| 要求龄期 | 要求检测的龄期天数 |
| 到期日期 | 自动计算 |

### 2.2 多龄期设置

部分样品需要多个龄期检测：

| 龄期 | 到期日期 |
|------|----------|
| 7天 | 成型日期 + 7天 |
| 28天 | 成型日期 + 28天 |

## 三、龄期计算服务

### 3.1 到期日期计算

```python
def calculate_due_date(molding_date: date, age_days: int) -> date:
    """计算到期日期"""
    return molding_date + timedelta(days=age_days)
```

### 3.2 龄期状态计算

```python
def get_age_status(molding_date: date, age_days: int, current_date: date) -> str:
    """获取龄期状态"""
    due_date = calculate_due_date(molding_date, age_days)
    days_remaining = (due_date - current_date).days
    
    if days_remaining < 0:
        return 'overdue'  # 已超期
    elif days_remaining <= 3:
        return 'urgent'   # 紧急（3天内）
    elif days_remaining <= 7:
        return 'warning'  # 预警（7天内）
    else:
        return 'normal'   # 正常
```

## 四、龄期预警

### 4.1 预警级别

| 级别 | 条件 | 标记颜色 |
|------|------|----------|
| 正常 | 剩余天数 > 7 | 绿色 |
| 预警 | 剩余天数 ≤ 7 | 黄色 |
| 紧急 | 剩余天数 ≤ 3 | 橙色 |
| 超期 | 已超过到期日期 | 红色 |

### 4.2 预警通知

系统自动发送龄期预警通知：
- 到期前7天：发送预警通知
- 到期前3天：发送紧急通知
- 到期当天：发送到期通知

### 4.3 通知接收人

- 检测人员：负责检测的人员
- 技术负责人：监督检测进度

## 七、常见问题

### Q1: 龄期计算是否考虑节假日？

**答案**：龄期按自然日计算，不考虑节假日。

### Q2: 样品超期后还能检测吗？

**答案**：可以检测，但需在报告中注明超期情况。

### Q3: 如何修改龄期设置？

**方法**：编辑样品信息，修改龄期要求。

### Q4: 多龄期样品如何管理？

**方法**：系统会为每个龄期生成独立的检测任务。
