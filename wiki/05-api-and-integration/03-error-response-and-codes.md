# 错误响应与业务码

HTTP 状态码表示传输层与框架层结果；业务成功与否常由 JSON 体内的 **`code`** 字段表示（与项目统一约定一致）。

注意：网关可能将 `code` 从数字变为 **字符串**，前端必须兼容（见 [前端信封](04-frontend-api-envelope.md)）。

既有文档：[API 约定与信封](../05-api-docs/01-api-conventions-and-envelope.md)
