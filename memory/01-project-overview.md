# 项目概览

## 基本信息

- **系统名称**：LIMIS（Laboratory Information Management Information System）
- **服务项目**：浦东国际机场四期扩建工程
- **机构性质**：工程第三方检测单位（驻场工地试验室）
- **工程预算**：588,199万元
- **工程周期**：2023年12月开工 — 2028年5月竣工
- **系统需持续服务至项目竣工**

## 核心定位

为第三方检测机构在浦东国际机场四期扩建工程设立的工地试验室提供全流程数字化管理，涵盖：
- 委托受理 → 样品管理 → 检测执行 → 报告出具的完整业务链
- 仪器设备、人员资质、环境监控等资源管理
- 质量管理体系（内审、管评、不符合、能力验证）电子化
- 统计分析与数据看板

## 合规要求

| 标准/制度 | 说明 |
|-----------|------|
| GB/T 27025-2019 | 检测和校准实验室能力的通用要求（等同ISO/IEC 17025:2017） |
| CMA资质认定 | 检验检测机构资质认定，报告须加盖CMA标识 |
| JGJ/T 233-2011 | 建筑工程检测信息管理规程 |
| GB/T 8170 | 数值修约规则与极限数值的表示和判定 |
| 见证取样制度 | 建设工程质量检测中的见证取样和送检制度 |
