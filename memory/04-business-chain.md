# 核心业务链

## 完整业务流程

```
工程项目登记 → 委托受理 → 合同评审 → 样品接收登记 → 样品编号标识
    → 任务分配 → 检测执行 → 原始记录填写 → 结果计算判定 → 记录复核
    → 报告编制 → 报告审核 → 报告批准 → 报告发放归档 → 样品处置
    → 统计分析
```

## 各环节数据流与状态机

### 委托单状态
```
草稿(draft) → 待评审(pending_review) → 已评审(reviewed) → 已退回(rejected)
                                         ↓
                                    已取消(cancelled)
```

### 样品状态
```
待检(pending) → 检测中(testing) → 已检(tested) → 留样(retained) → 已处置(disposed)
                                                                    ↑
                                                    已退还(returned) ─┘
```

### 检测任务状态
```
待分配(unassigned) → 待检(assigned) → 检测中(in_progress) → 已完成(completed)
                                                              ↓
                                                         异常(abnormal)
```

### 原始记录状态
```
草稿(draft) → 待复核(pending_review) → 已复核(reviewed) → 已退回(returned)
```

### 报告状态
```
草稿(draft) → 待审核(pending_audit) → 待批准(pending_approve) → 已批准(approved)
                 ↓                        ↓                        ↓
            退回编制(returned)       退回审核(returned)          已发放(issued)
                                                                   ↓
                                                              已归档(archived)
                                                                   ↓
                                                              已作废(voided)
```

## 关键业务规则

### 见证取样
- 法定检测项目必须见证取样
- 见证人由监理单位指派，需记录见证人姓名和证件号
- 见证比例需满足相关规定要求
- 系统自动统计见证比例

### 盲样管理
- 检测人员看到的样品编号不包含委托方信息
- 通过盲样编号与实际编号的映射表实现
- 映射表仅样品管理员和技术负责人可查看

### 龄期管理（混凝土试块）
- 标准养护试块：3d / 7d / 28d / 60d / 90d
- 同条件养护试块：等效养护龄期（累计温度≥600°C·d）
- 系统根据成型日期自动计算到期日并提醒

### 报告三级审批
- 编制人：填写检测数据，生成报告草稿
- 审核人：核查数据准确性、方法合规性（须为报告审核人角色）
- 批准人：最终签发（须为授权签字人角色，且授权范围覆盖该检测项目）
- 任一环节可退回，退回须填写意见

### 不合格结果处理
- 检测结果不合格时自动标记并通知技术负责人
- 不合格报告按正常流程签发
- 如需复检，须重新委托
- 不合格结果统计纳入合格率分析

## 检测项目分类（机场工程）

### 建筑材料检测
| 分类 | 检测项目 | 标准 |
|------|---------|------|
| 水泥 | 凝结时间、安定性、胶砂强度、细度 | GB/T 17671, GB/T 1346 |
| 钢筋 | 拉伸（屈服/抗拉强度、伸长率）、弯曲、重量偏差 | GB/T 228.1, GB 1499 |
| 混凝土试块 | 抗压强度、抗折强度 | GB/T 50081 |
| 砂 | 细度模数、含泥量、泥块含量、级配 | JGJ 52 |
| 碎石 | 压碎指标、针片状含量、级配、含泥量 | JGJ 52 |
| 外加剂 | 减水率、凝结时间差、抗压强度比 | GB 8076 |
| 粉煤灰 | 细度、需水量比、烧失量、含水量 | GB/T 1596 |
| 防水材料 | 拉伸强度、断裂伸长率、不透水性 | 对应产品标准 |
| 砌块/砖 | 抗压强度 | GB/T 2542 |

### 混凝土性能检测
- 拌合物：坍落度、含气量、温度
- 力学性能：抗压强度（标养/同条件）、弯拉强度
- 耐久性：抗渗等级、抗冻等级、抗氯离子渗透（电通量法）

### 地基与桩基检测
- 地基承载力：平板载荷试验
- 桩基完整性：低应变反射波法
- 桩基承载力：单桩静载试验、高应变法（CASE法/实测曲线拟合法）
- 基坑监测：水平位移、沉降、地下水位

### 钢结构检测
- 焊缝：超声波探伤、磁粉探伤
- 高强螺栓：扭矩系数、抗滑移系数
- 涂层：防腐涂层干膜厚度、附着力

### 机场专项检测
- 屋面抗风揭性能
- 道面混凝土弯拉强度（梁式试件）
- 跑道表面摩擦系数
- 跑道平整度（IRI）
- 沥青混合料：马歇尔稳定度、流值、空隙率、车辙试验动稳定度
