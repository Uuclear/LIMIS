# Superpowers Skills

This project uses the Superpowers skills library for structured development workflows.

## Available Skills

All skills are located in `/root/.claude/plugins/marketplaces/superpowers-marketplace/skills/`

### Core Skills

- **brainstorming** - Design refinement before coding
- **test-driven-development** - RED-GREEN-REFACTOR cycle
- **systematic-debugging** - 4-phase root cause analysis
- **using-git-worktrees** - Isolated development branches
- **writing-plans** - Implementation planning
- **subagent-driven-development** - Parallel agent workflows
- **requesting-code-review** - Pre-review checklist
- **finishing-a-development-branch** - Merge/PR workflow

## Usage

Before starting any task, check if a relevant skill applies and invoke it using the `Skill` tool.

Reference: `/root/.claude/plugins/marketplaces/superpowers-marketplace/skills/using-superpowers/SKILL.md`
